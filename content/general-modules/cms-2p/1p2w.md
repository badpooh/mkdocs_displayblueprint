# CMS-2P 1P2W

`1P2W`는 CMS-2P의 결선 설정 기준 화면 묶음이다. 이 설정 안에서 HOME Dashboard 등 세부 화면으로 이동한다.

## 화면 구성

| 분류 | 화면 |
| --- | --- |
| HOME Dashboard | [MAIN](home-dashboard-1p2w-main.md), [cms-2p](home-dashboard-1p2w-module.md) |

## 적용 기준

- 1P2W 설정에서는 단일 피더 기준으로 화면 항목을 표시한다.
- 화면별 세부 동작은 각 화면 문서에서 관리한다.
- 설정 조건에 따라 같은 화면이 달라지는 상태는 해당 화면 문서의 조건부 상태에서 관리한다.
