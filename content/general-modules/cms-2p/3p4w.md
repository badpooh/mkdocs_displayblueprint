# CMS-2P 3P4W

`3P4W`는 CMS-2P의 결선 설정 기준 화면 묶음이다. 이 설정 안에서 HOME Dashboard 등 세부 화면으로 이동한다.

## 화면 구성

| 분류 | 화면 |
| --- | --- |
| HOME Dashboard | [MAIN](home-dashboard-3p4w-main.md), [cms-2p](home-dashboard-3p4w-module.md) |

## 적용 기준

- 3P4W 설정에서는 페어링 구성의 Dashboard 항목을 표시한다.
- 3P4W의 MAIN 화면은 3P3W MAIN 화면 자산을 공통으로 사용하되, 모듈 전환 시 3P4W cms-2p 화면으로 이동한다.
- 화면별 세부 동작은 각 화면 문서에서 관리한다.
- 설정 조건에 따라 같은 화면이 달라지는 상태는 해당 화면 문서의 조건부 상태에서 관리한다.
