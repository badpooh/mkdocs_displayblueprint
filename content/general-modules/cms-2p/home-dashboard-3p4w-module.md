## 적용 정책

- 좌측·우측 화살표 중 어느 쪽을 눌러도 3P4W MAIN Dashboard로 이동한다.
- 3P4W 페어링 구성은 상단 모듈선택창에 마스터·슬레이브 ID를 함께 표시한다.

<hr class="section-divider">

## 화면 동작

<p><span class="marker-ref">1</span><strong>Pairing Module ID 표시</strong></p>

- 「분기 > SETUP > MEASUREMENT > Load Mode > Pairing Module ID」의 설정값을 상단 모듈선택창에 표시한다.
