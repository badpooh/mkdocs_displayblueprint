| `Policy-23-0012` | 전압/전류/ZCT 색상 정책 |
| `Policy-23-0016` | Description 문자열 입력값이 없을 때 빈칸 표시 |

| `Policy-23-0032` | EVENT 이벤트 리스트의 `Read All` 표시 및 동작 정책 |

| `Policy-23-0047` | `EVENT > MEASUREMENT`의 `All`/`Etc.` 부메뉴 정책 |
| `Policy-23-0049` | 데이터뷰 제목 Full Name 및 Residual Voltage 제목 정책 |
| `Policy-23-0052` | `DO Notification` 선택 채널 및 모듈 탈락 표시 정책 |
| `Policy-24-0027` | Description 문자열 최대 표시 `20자+…` 및 고정폭 폰트 정책 |
| `Policy-24-0033` | EVENT 이벤트 테이블의 시간 필드명 `Time` 정책 |
