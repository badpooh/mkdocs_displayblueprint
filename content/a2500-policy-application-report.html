적용 기준

  • A2500 MAIN/HOME/SETUP 화면에 직접 연결되는 정책만 문서에 삽입했다.
  • CMS-2P 전용 조건, 특정 제품군 전용 조건처럼 A2500 화면에 바로 대응하지 않는 항목은 제외했다.
  • 기존 화면 설명과 기존 화면별 정책은 보존하고, 새로 삽입한 내용은 codex-policy-start / codex-policy-end 마커 안에 넣었다.
  • 소스 이미지 칸은 만들지 않았고, 정책 근거만 코드 형태로 표시했다.
