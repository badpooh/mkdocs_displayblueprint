## 안건: [A2550ZMM] 모듈 설계 방향 공유

- 기본 컨셉: CMM 모듈에 누설전류 계측 추가 모듈(ZM Miniature). ZM은 CMS Sub모듈이나 ZMM은 Sub모듈 아님.

> [이미지 Untitled 13.png] [참고/손그림] ZMM 구성 — MCCB·CMM·LCD 배치, ZM(LCF Y→O, DI →X, Comm RJ12→RJ45)과 3MM(ZMM) 비교.

- 상세모델명 "A2550ZMM-3F-60A". A2550ZM-1P3F-60A와 유사 형태(새 금형). 비교: ZM은 LCD·Button·통신ID 없음(CMS 종속, RJ12 1ea), ZMM은 LCD·Button·통신ID 있음(A2500 통신, RJ45 2ea), 둘 다 DI 6ch. 분류: 특수모듈(Leakage만 계측, CM계열에 전달, 주메뉴/부메뉴 구성이 CM계열과 달라 특수모듈).
- 「HOME」: [ZMM] 분기는 「MEASUREMENT > Leakage」에서 Feeder 3개 값(Min/Max, Off 피더 Hidden). [CMx+ZMM] 분기는 연결모듈에 따라 표시.
- 「SETUP」: 연결 설정 주체 [ZMM](Feeder별 연결 모듈 ID 선택, 모든 Feeder Off 가능 — 현장 초기설치 고려). 연결 가능 모듈: Leakage 계측 없는 CM계열(**[CMM-3P_3Pmode]만 연결설정 가능**, CMZ·CMS·CM-3P-400A+ZCT 등은 자체 Leakage라 불가, 60A 이하만). 「CMx+ZMM > SETUP > EVENT > Leakage」 부메뉴 표시 → **1안(미표시)** 결정.
- 「EVENT」: [ZMM] 분기는 「MEASUREMENT > Leakage」 이벤트리스트. 「CMx+ZMM」 부메뉴 표시 → **1안(미표시)** 결정.

> [이미지 Untitled 14.png] [참고/A2550CM·CMZ Application 정리 20230713 변영복] Current Measurement(1P/3P3W/3P4W) CM 800A~60A, Current/Leakage(CMZ) 800A~60A, 3P4W(CM+외부ZCT·CMS-3P4W+ZM·CMM-3F+ZMM). 적용 MCCB Type·전선 굵기·분전반 설치조건 복합 검토 필요.

**👉 「HOME」에만 누설값 표시, 「SETUP」·「EVENT」는 [ZMM]에서만 설정/확인 가능** ([CMM-3P_3Pmode]만 연결설정 가능)
