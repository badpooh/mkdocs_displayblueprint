# 정기회의-20230518

## 안건: 「A2550CM-3P-400A/630A/800A+외부ZCT」 "외부ZCT 사용" 항목명 확정 (`정기회의-20231019` 일부 수정)

- 「정기회의-20230511」 재논의. 항목명 후보 검토(강경아, 2023/05/12): Use of External ZCT / External ZCT Usage / Operation / Operating / Connection / Connecting 등. Tobe: **"External ZCT Connection"**(0 Disable 디폴트 / 1 Enable).

> [이미지 Untitled.png] [Tobe/ID01 CM-3P-400A+Z] SETUP > MEASUREMENT > Current — Rated Current 60, TDD Reference Selection Peak Demand Current, TDD Nominal Current [A] Rated Current, "External ZCT Connection" Enable, ZCT Trend Data Type RMS, ZCT Min. Measured Current [mA] 10.

- "External ZCT Connection" 설정(Disable/Enable)에 따라 「ZCT Trend Data Type」·「ZCT Min. Measured Current [mA]」가 Disabled/Enabled.

**👉 Tobe 안으로 결정**

## 안건: 「SETUP」 변경 항목 Highlight 표시 정책 보완 (`정기회의-20230525` 결정)

- 「정기회의-20230330」 결정(변경 항목 Highlight)의 상황별 처리 보완.

> [이미지 Untitled 1.png] [참고] SETUP > Voltage — "Wiring 3P4W" 변경 Highlight(짙은색), Apply/Cancel 표시.

- [이슈-1] Highlight 항목을 Apply 전 다시 기존 저장값으로 되돌린 경우. Asis: A2500은 저장값=현재값이면 변경 처리 안 함, A2700·A3700N·AccuraSM은 한번이라도 변경하면 변경 처리. Tobe — 1안(저장값과 현재값 비교, 다를 때만 Highlight, A2500 방식) vs 2안(한번이라도 변경 시, A3700N 방식). 투표 결과 총21표 중 1안 13표(61.9%)·2안 8표(38.1%).

**👉 [이슈-1] "1안"으로 결정** (AccuraSM은 예외, 현 상태 유지)

- [이슈-2] Highlight 항목이 Apply 전 다른 항목에 의해 Disabled되는 경우. 예: Load Mode "3-phase"→"1-phase"로 Feeder1 Phase Selection Enabled→"NA"로 변경→다시 "3-phase"로 Feeder1 Disabled.

> [이미지 Untitled 2.png] [예시/ID00 CM-3P-60A] SETUP > MEASUREMENT > Load Wiring — Load Mode "3-phase mode", Feeder1/2/3 Phase Selection AN/BN/CN(회색 Disabled).

> [이미지 Untitled 3.png] [Asis/A3700N·A7300] SETUP > PROTECTION > OCR1(Inv.) — 변경Highlight(백색 글자+Highlight 바탕, "Time Pickup Level [A] 0.50"), Disabled(회색 글자+바탕 유지, "Inst. Pickup Level"·"Operate Time Delay").

> [이미지 Untitled 4.png] [Asis/A2500] Load Wiring — 변경Highlight(글자 유지+Highlight 바탕), Disabled(글자 유지+회색 바탕). 2개 상황 동시 표시 불가.

- Tobe 공통: Disabled돼도 사용자 변경값 유지(예: Feeder1 "NA" 유지), Apply/Cancel 표시. 1안(변경Highlight+Disabled 동시 표시, A3700N은 회색글자+Highlight바탕 가능, A2500은 동시 표시 불가) vs 2안(Disabled만 표시). 투표 결과 총21표 중 1안 14표(66.7%)·2안 7표(33.3%).

**👉 [이슈-2] "1안"으로 결정**

## 안건: 「SETUP」 버튼 디자인 변경

- 위 Highlight 정책 "1안"을 반영한 디자인시안. 조국춘 의견: A2500 기본 바탕색을 연회색으로(변경Highlight+Disabled 동시 표시 여지), 부메뉴 하단(Blank)을 회색(A3700N처럼). Disabled 성격을 "Disabled"(현재 의미 없음)와 "ReadOnly"(의미 있음, 예 「NETWORK > MAC Address」)로 구분.

> [이미지 Untitled 5.png] [Tobe/시안1 기존] SETUP 버튼 상태 — Normal/Pressed/Disabled/Disabled&Pressed/Read Only/Read Only&Pressed(기존 디자인).

> [이미지 Untitled 6.png] [Tobe/시안2-1] A3700N 컨셉 적용 — 기본 바탕 연회색·부메뉴 하단 회색, "Disabled&변경Highlight"·"ReadOnly&변경Highlight" 바탕 짙은회색.

> [이미지 Untitled 7.png] [Tobe/시안2-2] 동일하나 "Disabled&변경Highlight"·"ReadOnly&변경Highlight" 바탕을 연한청색으로 처리.

- 4가지 상태 구분: "Disabled", "ReadOnly", "Disabled&변경Highlight", "ReadOnly&변경Highlight"("Pressed"는 Tap 중 표시, 변경Highlight와 같은 형태).

**👉 Tobe "시안2-2"로 결정**

## 안건: 「SETUP」 값이 다른 항목에 의해 자동 변경되는 경우 처리 (`정기회의-20230615` 재논의 미결정, `정기회의-20230706` 결정)

- 예: 3P3W에서 「Reference Mode」 "Line to Line"(ReadOnly), 3P4W로 변경 시 Enabled→"Line to Neutral" 변경→다시 3P3W로 ReadOnly되며 "Line to Neutral" 유지→Apply 시 "Line to Line"으로 자동 변경. ReadOnly 시점과 값 변경 시점이 달라 문제.

> [이미지 Untitled 8.png] [예시] SETUP > MEASUREMENT > Voltage — Wiring 3P4W, Reference Mode "Line to Line"(3P3W는 ReadOnly만, 3P4W는 Line to Line·Line to Neutral 선택).

- Tobe — [이슈-1] ReadOnly로 변경될 때 값도 같이 변경(3P3W 복귀 시 "Line to Line" 자동), 이후 3P4W 변경해도 값 유지(사용자가 다시 변경). [이슈-2] 다른 항목에 의해 값 자동 변경+ReadOnly된 경우 "변경Highlight"+"ReadOnly" 동시 표시.

**⏭️ 추가검토필요** (PNT팀 발족 이전 기존 정책 — UI단 FitToRange·Formatting 안 하고 Apply 시 MCU단 자동 변경 — 과 상반되어 재검토)

## 안건: 「Voltage > Reference Voltage Mode」 결선모드별 범위·속성 수정 (`정기회의-20230615` 재논의 미결정, `정기회의-20230706` 결정)

> [이미지 Untitled 9.png] [Asis/Tobe] SETUP > MEASUREMENT > Voltage — Wiring 3P4W, PT Primary/Secondary 380, "Reference Voltage Mode" Line to Line(항목명 "Reference Mode"→"Reference Voltage Mode" 검토).

- Asis 결선모드별 「Reference Voltage Mode」: 3P4W(Line to line/Neutral, Enable), 3P3W(Line to line, Disable), 1P2W(Line to line/Neutral, Enable), 1P3W(Enable, A2500 설정불가·A3700N 가능). 1P2W는 Line to Line/Neutral 값 동일하나 현장에서 "Line to Line" 표시가 어색.
- Tobe: 3P3W를 **ReadOnly(Line to line 고정)**, 1P2W를 **ReadOnly(Line to Neutral 고정)**로 수정(값 고정이나 「Reference Primary Voltage [V]」에서 참조됨).

**⏭️ 추가검토필요** (위 자동 변경 안건과 동일 이슈)
