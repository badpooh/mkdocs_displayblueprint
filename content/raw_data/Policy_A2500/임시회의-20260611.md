- 추후 A2550CMS-2P에도 동일한 Hysteresis 정책을 적용한다.

**👉 결정사항**

- A2500D에도 「Remote Reset」 부메뉴를 표시하며 설정항목명도 `Remote Reset`으로 한다.

- 1:Enable(디폴트)은 「메인 > SETUP > SYSTEM > LED」의 `EVENT LED Hold Time [sec]` 설정 또는 모듈 SET 버튼에 따라 이벤트를 리셋한다.

## 안건: [04] A2500M과 A2500D CPU Application Version 불일치 팝업 수정

- 기존 전체 화면 버전 불일치 팝업에는 IP Address가 표시되지 않아 팝업이 열린 상태에서 네트워크 업데이트를 진행하기 어려운 문제가 있었다. 팝업은 버전 업데이트로만 해제되고 물리버튼 화면 전환도 허용하지 않는다.

> [이미지 Untitled.png] [참고/기존 CPU App. 버전 불일치 팝업] A2500M과 A2500D의 버전 통일을 유도하는 전체 화면 팝업. 버전은 앞 세 자리만 표시하고 Develop 버전은 표시하지 않음.

> [이미지 image 43.png] [참고/A2500 NETWORK Ethernet] IP Address·Subnet Mask·Gateway 확인 화면.

### Asis

> [이미지 image 44.png] [Asis/버전 불일치 팝업] IP Address 정보가 없어 팝업 표시 중 네트워크 업데이트가 어려움.

### Tobe 검토안

> [이미지 image 45.png] [Tobe 1안/IP Address 표시] 팝업에 IP Address만 추가.

> [이미지 image 46.png] [Tobe 2-1안/네트워크 정보 표 표시] IP Address·Subnet Mask·Gateway를 표로 표시.

> [이미지 image 47.png] [Tobe 2-2안/네트워크 정보 텍스트 표시] IP Address·Subnet Mask·Gateway를 텍스트로 표시.

> [이미지 image 48.png] [Tobe 3안/닫기 버튼 추가] 닫기 버튼으로 팝업을 해제하고 30분 주기로 다시 표시.

**👉 결정사항**

- Tobe 2-1안으로 결정한다. 버전 불일치 팝업에 IP Address·Subnet Mask·Gateway를 표로 표시한다.
- 팝업은 버전 업데이트로만 해제하며 물리버튼 화면 전환을 허용하지 않는다.
- 업데이트 시 버전 Mismatch 팝업 위에 업데이트 팝업을 표시한다.
