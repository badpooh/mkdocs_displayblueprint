# 임시회의-20240912 (긴급)

본 회의는 "A2500D Renewal(1.11.0) Test - 4차" 진행에 앞서 구현 결정이 필요한 사항(구현 여부·구현 버전)을 논의하는 긴급회의.

## 안건: 「분기 > SETUP > EVENT > Leakage」 모듈별 이벤트 Off 조건 (2차 테스트)

- 2차 테스트 결함: Leakage 이벤트 On 상태에서 모듈 탈락 시 Event Off 미동작, 재판정 시 동작하나 Cause가 "Due to setup changed"로 표현됨. Asis: [CMZ]에서 Leakage 이벤트 On 중 CMZ 탈락 시 "Due to setup changed" 동작.
- Tobe(Off 조건 구분): [CMZ] — CMZ 탈락 "Due to module disconnected", Leakage 설정 변경 "Due to setup changed". [CM3P+외부ZCT] — CM3P 탈락 "Due to module disconnected", External ZCT Connection 설정 변경·Leakage 설정 변경 "Due to setup changed". [CMS+ZM·CMDR+RM] — CMS/CMDR 탈락 "Due to module disconnected", ZM/RM 탈락 "Due to Submodule disconnected", Leakage 설정 변경 "Due to setup changed".

**👉 Tobe 안으로 결정** (적용 시기: v1.11.0 이후)

## 안건: 「A2550TEMP > 분기 > SETUP > I/O > DO」 Pulse Period·Pulse On Time 설정 범위 (3차 테스트)

- 3차 테스트 결함: "Pulse Period"·"Pulse On Time" 설정 범위가 설계서와 상이. Asis — 설계서 0.02~20.00/0.01~20.00, Renewal D 0.2~9.9/0.1~9.9(TEMP는 100ms 해상도라 범위 수정 불가). 의존성 현황: A2550IO-DO 의존성 제거, A2550TEMP·DCM 의존성 존재(Pulse Period 변경 시 Pulse On Time 변경, Pulse On Time은 Period 초과 불가).
- Tobe: 설계서·장치 범위 변경(Pulse Period "0.2~20.0", Pulse On Time "0.1~20.0"), Pulse Period·Pulse On Time 의존성 제거.

**👉 Tobe 안으로 결정** (단, 의존성 제거는 v1.11.0 이후 적용)

## 안건: 「A2550TEMP > 분기 > HOME > I/O > DI」 DI 상태 표시 기준 (3차 테스트)

- 결함: A2550IO-DI(Physical Status)와 A2550TEMP(Logical Status) 간 DI Open/Closed 표시 기준 상이. 참조: 「Policy-23-0043」(장치화면에서는 "Physical Status"만 표시).

> [이미지 image.png] [참조/A2550TEMP DI Modbus Map] 64 "Digital input state"(UINT16, R) — DI Polarity에 따라 달라짐(Normal: 0 Open/1 Close, Reverse: 0 Close/1 Open). 65 "Digital input on hold"(과거 DI on 보조지표, on 시 1·60초 후 0 clear, Normal Polarity만 동작·실제 물리값), 66 "Digital input off hold", 67 "Digital input number of pulse"(DI 발생 횟수).

- Asis: A2550IO-DI는 Physical Status, A2550TEMP는 Logical Status 표시. Tobe: A2550TEMP 화면도 DI 상태를 Physical Status로 표시.

**👉 Tobe 안으로 결정** — Modbus Map의 Digital Input State를 "Physical Status" Address로 변경(적용 시기: v1.11.0).

> [이미지 image 1.png] [참조/Under Event Off, MAIN] EVENT > MEASUREMENT > All — Current Under Off(Phase A/B/C, Cause: Due to out of range), Under Start(Start Current 0.007/0.008/0.058 [A]).

> [이미지 image 2.png] [참조/Under Event End, MAIN] EVENT > MEASUREMENT > All — Current Under End(Phase A/B/C, Peak Current 0.000 [A]·Duration 10.1 [s]), Under Start(Start Current 1.246 [A]).

- Asis: Under 이벤트 On 상태에서 Low Limit 조건 만족 시 이벤트 Off("Due to out of range"). Tobe: Low Limit 조건 만족 시 이벤트 End로 구분 변경.

**👉 Tobe 안으로 결정** (적용 시기: v1.11.0 이후)

## 안건: 「분기 > SETUP > EVENT > Current」 Under 이벤트의 Low Limit에 따른 이벤트 동작

> [이미지 image 3.png] [참조/A7300 UVR 항목] UVR1/UVR2 항목 리스트 — Function(0 Disable/1 Trip/2 Control), Voltage Selection, Operate Curve Type, L-L Pickup Level 30.0~190.0(디폴트 90.0)·L-N Pickup Level 15.0~110.0(디폴트 50.0), Time Multiplier, Min. Operating Voltage(Vmin) 0~85 디폴트 11(최소 동작전압, 이하면 보호계전 미동작·0이면 Disable·반드시 픽업 레벨보다 낮아야 함), Reset Time Delay.

> [이미지 image 4.png] [참조/A7300 UVR 블록도] UVR 동작 블록도 — VA(or VAB) → Level Detector(V < Gs, V·IVmin) → & → DT or IDMT(Top/Trs) → PICKUP/OPERATE, CB Condition·CB ON Status·FF_OP·BLOCK.

- Asis: Low Limit 값이 Threshold보다 크면 이벤트 Start 후 즉시 Off("Due to out of range"). Tobe: Low Limit > Threshold면 이벤트 Start 미동작(A7300 UVR과 동일).

**👉 자사 제품 동일 기능(Low Limit, Min. Operating Voltage) 동작 현황 확인 후 재검토**

## 안건: 「분기 > SETUP > I/O > DO」 모듈별 DO Control 설정 범위와 Control 동작

- 결함: A2550IO-DO(Control 범위 "Off/On")와 A2550TEMP·A2550DCM(Control 범위 "None/Operate") 간 상이. A2550DCM은 Polarity Reverse 시 DO Control 미동작. 「임시회의-20240530」에서 DO Control 디폴트 추가·A2550 모듈 모두 수정은 "추후 재검토" 상태.

(별도 결정 없음 — Asis 현황 공유)

## 안건: 「CM-3P+외부ZCT」 모듈의 「Load Mode」 화면 표시 여부 재검토

- 「임시회의-20240328」·「임시회의-20230511」 설계수정 안건. 「임시회의-20240328」 결정: [CM-3P-5A+외부ZCT]는 「Load Mode」 부메뉴 삭제·「CT Ratio」 유지.

> [이미지 Untitled.png] [참고/CM-3P-5A+외부ZCT SETUP Current, ID00] SETUP > MEASUREMENT > Current — Rated 400·Min. Measured 0.10%·TDD·CT Ratio 1(「Load Mode」 부메뉴 삭제, 「임시회의-20240328」 자료).

> [이미지 Untitled 1.png] [참고/CM-3P-5A+외부ZCT SETUP Leakage, ID00] SETUP > MEASUREMENT > Leakage Current — Trend Data Type Fundamental·Min. Measured 1·External ZCT Burden Resistor 1,200·External ZCT Internal Resistor 1,000·External ZCT Connection Disable.

> [이미지 Untitled 2.png] [참고/Leakage 항목범위] Leakage — Trend Data Type(0 RMS/1 Fundamental), Min. Measured Current [mA] 0~100, External ZCT Burden Resistor [Ω] 1~9,999 디폴트 1,200(CM-3P-5A/400A/630A/800A+외부ZCT인 경우만 표시), External ZCT Internal Resistor [Ω] 1~9,999 디폴트 1,000, External ZCT Connection 0 Disable/1 Enable.

- Asis: [CM-3P+외부ZCT] 400A 이상·5A 모듈은 「Load Mode」 부메뉴 장치화면에서만 삭제(통신·모듈에서는 1-Phase Mode 설정 가능). 통신에서도 삭제 시 기존 [CM-3P-5A] 1-Phase Mode 현장 대응 어려움([CM-3P-5A]와 [CM-3P-5A+외부ZCT] 구분 방법 없음).
- Tobe(「임시회의-20250819」 자료 참고): ① 하드웨어 버전 읽어 [CM-3P]와 [CM-3P+외부ZCT] 구분, ② 외부ZCT 설정에 따라 「Load Mode」 부메뉴 Shown/Hidden(Policy-23-0038 따름, Enable 시 숨김·Disable 시 표시).

**👉 결정사항** — (1) 하드웨어 버전 읽지 않고 [CM-3P+외부ZCT] 모듈(5A/400A/630A/800A)은 「Leakage」 부메뉴 표시. (2) 외부ZCT 설정(External ZCT Connection)과 무관하게 「Load Mode」 부메뉴 항상 표시(숨기지 않음). 단, 외부ZCT "Enable"이어도 「Load Mode」가 "1-Phase Mode"면 「HOME」은 CM-3P_1Pmode와 동일 표시("Leakage" 관련 항목 미표시). 적용시기: v1.11.0 이후.
