## 안건: [02] [CMS-2P] ZM 연결 상태와 외부 ZCT 설정 상태에 따른 동작 방안

- 「임시회의-20250930」에서 결정. [CMS-2P]는 ZM 또는 외부ZCT로 누설전류 계측. [CM-3P+외부ZCT]는 외부ZCT 연결 무관하게 「Leakage」 부메뉴·설정 표시·External ZCT Connection 설정에 따라 계측값 표시. [CMS+ZM]은 A2550ZM 연결여부에 따라 「Leakage」 부메뉴·계측값 표시.

> [이미지 image 19.png] [참고/CMS+ZM SETUP Leakage, ID03 CMS-1P3F+ZM] SETUP > MEASUREMENT > Leakage Current — Trend Data Type Fundamental·Min. Measured Current [mA] 1(A2550ZM 연결 시 「Leakage」 부메뉴 표시).

> [이미지 image 20.png] [참고/CMS-1P3F Current, ID02 CMS-1P3F-60A] SETUP > MEASUREMENT > Current — Feeder1~3 탭, Rated 60·Min. Measured 0.10%·TDD*(ZM 미연결 시 Leakage 부메뉴 없음).

> [이미지 image 21.png] [참고/CMS+ZM HOME Dashboard, ID03 CMS-1P3F-60A+ZM] HOME > Dashboard — VOLTAGE Phase-A 220.1V·CURRENT 29.96A·Leakage(Fund.) 2mA·POWER 5.709kW·ENERGY 222,474,457kWh·SYSTEM Svr. Rack#31(A2550ZM 연결 시 누설전류 계측값 표시).

- Tobe: ZM·외부ZCT 연결/설정 무관하게 누설전류 관련 설정 항목 「SETUP > MEASUREMENT > Leakage」 표시. 1안) ZM 설정 추가("External ZCT Connection"→"ZCT Connection", 범위 0 Disable 디폴트/1 ZM/2 External ZCT; ZM 시 연결여부 고려 표시·미연결 "-"·CMS↔CMS+ZM 표기, External ZCT는 연결 무관 표시). 2안) ZM 설정 미추가(ZM 연결 시 [CMS+ZM] 동일·미연결 시 [CM-3P+외부ZCT] 동일).

**👉 결정사항** — 추후 재검토. 회의내용: 2안 우선순위는 ZM(External ZCT Connection Enable이어도 ZM 연결 상태 우선 — ZM 연결 시 ZM 계측·미연결 시 External ZCT 계측). 하드웨어측 External ZCT 연결 상태 장치 확인 가능 여부 확인 후 추가 검토.
