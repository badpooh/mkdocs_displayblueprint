## 안건: 「상단바 > 모듈선택창 ◀/▶」 모듈 이동 순서

- Asis(v1.5.9 개발): Main → CM3P → CM1P → CMS 1P3F → CMS 1P2F → CMS 3P4W → TEMP → GW → DCM. 동일 종류 내 ID 오름차순. CM3P "1-phase mode"는 CM3P로 동일 취급. (「정기회의-20230420」 최종결정.)
- Tobe 공통: 특수모듈 DCM→TEMP→GW 순, 동일 계열 내 ID 오름차순. 1안: CM3P계열→CM1P계열→특수모듈. 2안: 3P/1P 구분 없이 CM계열로 처리. 3안: 데이터뷰 제공 모듈 내에서만 ID 오름차순(A2300 방식).

**👉 ID 오름차순 이동, 같은 모델이면 화면 Index 유지, 다른 모델이면 현재 모드 대시보드로 이동(영업의견 청취 필요)**
