# 임시회의-20240125

## 안건: 「분기 > A2550TEMP > SETUP > EVENT」 THSEN 관련 설정 추가

- THSEN(온습도 센서) 1개 이상 연결 시 습도 관련 부메뉴 표시. 이벤트 동작은 「Temperature」·「Diff. Temp.」과 동일(미연결 센서 이벤트는 미발생).

> [이미지 Untitled.png] [Asis/TEMP SETUP EVENT Temperature, ID10 TEMP+SENSOR6] SETUP > EVENT > Temperature(TEMP) — Trigger Disable·Threshold [℃] 40.0·Time Delay [ms] 0·DO Notification Disable(부메뉴 Temperature/Diff. Temp./Event Reset, 습도 부메뉴 없음).

> [이미지 Untitled 1.png] [Tobe/TEMP SETUP EVENT Humidity, ID10] SETUP > EVENT > Humidity(Sensor.1) — Trigger Disable·Threshold [%] 40.0·Time Delay [ms] 0·DO Notification Disable(부메뉴 Humidity·Diff. Humidity 추가). 항목 범위: Trigger 0 Disable/1 Enable, Threshold [%] 1.0~100.0 디폴트 40.0, Time Delay [ms] 0~10,000 디폴트 0, DO Notification 0 Disable/1 Enable.

> [이미지 Untitled 2.png] [Tobe/TEMP SETUP EVENT Humidity 콤보, ID10] SETUP > EVENT > Humidity — 센서 콤보 팝업 Sensor.1~6.

> [이미지 Untitled 3.png] [Tobe/TEMP SETUP EVENT Diff. Humidity, ID10] SETUP > EVENT > Diff. Humidity — "Difference Humidity (A-B)"·Trigger Disable·A Select Sensor Max·B Select Sensor Min·Threshold [%] 40.0·Time Delay [ms] 0·DO Notification Disable.

> [이미지 Untitled 4.png] [Tobe/Diff. Humidity 항목범위] Diff. Humidity — Trigger 0 Disable/1 Enable, A Select 1 Sensor.1~6 Sensor.6/7 Sensor Max(THSEN 중 가장 높은 습도, A·B Select 같은 값 불가), B Select 1~6/7 Sensor Min(가장 낮은 습도), Threshold [%] 0.0~100.0 디폴트 40.0, Time Delay [ms] 0~10,000, DO Notification 0 Disable/1 Enable.

- Main CM 아이콘 표시(「ChainView」)는 「임시회의-20251127」 자료 참고.

**👉 결정사항** — 「Temperature」는 연결된 센서만 표시, 「Humidity」는 TSEN이 연결된 경우 센서를 표시하지 않음. 「Diff. Temp.」·「Diff. Humidity」의 "A Select"·"B Select"는 미연결 센서를 항목값으로 설정 시 Invalid 표시 추가.

## 안건: 「Main A2550CM ID」 설정 시 「Chain View」에 Main CM 아이콘 표시 추가

- 「임시회의-20240111」 추가안건(「ChainView」 MainCM 장치 표시 추가). Main CM 아이콘 1안(아이콘 좌측 상단 별 아이콘)/2안(왕관 아이콘). 표시조건(AND): 「Main A2550CM Assignment」 Enable AND 「Main A2550CM ID」 ID 설정. 숨김조건(OR): Assignment Disable, 또는 경고 아이콘 표시 시(ID 충돌 등). 화면 예시는 「임시회의-20251127」 자료 참고.

**👉 Tobe 안으로 결정** (Main CM 아이콘 표시는 2안 왕관 아이콘)
