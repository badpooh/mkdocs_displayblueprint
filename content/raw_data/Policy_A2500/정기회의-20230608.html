정기회의-20230608

안건: 「SETUP > Event」 Function 항목명을 "Trigger"로 통일

  • Asis: Dip·Swell에서 타이틀과 동일 항목명 중복, Function 항목명이 메인·분기마다 상이(메인 Dip "Dip Function"·Swell "Swell Function"·Transient Volt. "Event Trigger" / 분기 Current·Leakage "Trigger"·Temperature·Diff. Temp. "Event Trigger").
  • Tobe: 이벤트 Function 항목명을 "Trigger"로 통일(설정항목 다수: Dip·Swell·Transient Volt.·Temperature·Diff. Temp. 등). 설정항목 1개면 이벤트명만 표시(SEMI~·IEC~·ITIC·Fuse Fail·Phase Open 등). Dip·Swell 타이틀 중복 항목명(Trigger/Threshold [%]/Hysteresis [%]) 삭제.

👉 Tobe 안으로 결정

  • Asis: 「Location」은 공통 설정항목(모든 피더탭 같은 값)이나 공통/개별 구분이 안 돼 버그로 오인 가능.

  • Asis: 「Load Mode」가 「Load Wiring」 내 항목.

[CMZ-3P]·[CMZ-3P4W]·[CMS-3P4W]도 불필요 항목(Load Mode·Feeder Phase Selection) 삭제. [CM-3P]도 동일 컨셉 수정.
