- Tobe: 「ID Setup」 버튼 추가(「SETUP > Chain View」만 표시·HOME/EVENT 미표시, Pressed 상태에서 화면 전환 시 해제). 설정변경팝업(Pressed 상태에서 모듈 Tap, A2500·Rotary Switch·Unknown은 동작 없음, 팝업 제목에 모듈 정보, 입력필드에 현재 ID, 「Enter」 후 「Apply」로 변경, Policy-24-0028 Clipping, "Standby ID" 체크 시 ID 255). ID 변경 모듈 아이콘 1안 주황색 테두리/2안 남색 테두리. (화면 예시는 「임시회의-20250219(긴급)」·「임시회의-20250123」 자료 참고.)

**👉 결정사항** — Tobe 안으로 결정. ID Setup 설정 적용 방식은 v1.11.0에서 1안(Apply 즉시 변경)으로 적용(추후 2안 검토). 회의내용: Entered 테두리 색상은 남색이 좋음(주황색은 ChainView Save 변경, 남색은 ID 설정 변경)이나 디자인팀은 기존 주황색 동일 의견. 일괄 변경(Apply/Cancel) 구현은 일정상 어려울 수 있음(1안·2안 클릭 수 동일).
