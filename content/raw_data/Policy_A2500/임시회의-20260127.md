# 임시회의-20260127

## 안건: [01] [A2550IO-DI], [A2550IO-DO] 채널별 "User-Defined Info" 추가 관련 검토사항

- 「임시회의-20260120」 [03] 설계 수정 안건. 「임시회의-20260120」 결정: A2500 Modbus Map에만 채널별 "User-Defined Info" 추가(모든 IO 모듈, IO-TEMP는 개발 검토 후), DejaVu 18pt 15자, AccuraSM은 계측·설정 모두 구현. 본 회의는 설정 글자수 축소 검토.

> [이미지 image.png] [참고/메일 문부기 2026-01-26] "A2550 IO 채널별 description size 축소 검토" — 채널별 description을 30byte(15word) → 10byte(5word)로 감소 검토. DI 1 개당 최대 22채널이므로 15→5word 시 Description 공간 330→110word(Module ID·access 번지 포함 총 332→112word). 기존 잔여 Modbus map 342word 중 332word 소비 시 잔여 10word, 112word 소비 시 잔여 230word(차후 활용 가능). 결론: 5word 감소 시 Modbus 공간 절약, 단 10글자 충분할지 고객 의견 수렴 필요.

- Asis: 채널별 "User-Defined Info" 설정 글자수 15자(8word).
- Tobe: 1안) 10자(5word) / 2안) 6자(3word).
