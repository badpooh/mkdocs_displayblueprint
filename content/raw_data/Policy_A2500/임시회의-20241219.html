- 회의 내용
    - Orange, Red 색상은 사용자에게 부정적으로 받아들여질 수 있음 (From. 문부기)
        - 판넬 안에 설치되는 장치로서 평상시 외부에서 확인되는 정보가 아님 (From 변영복)
    - 색상 일관성 측면에서 통신상태를 Orange로 표시하는 것이 일관성 유지면에서 좋을 것으로 보임 (From 김민수)
    - 해당 CMS-2P의 LED 용도가 설치과정에서의 문제 상황을 판단하기 위함인지? (From 윤우진)
        - 설치과정과 더불어 운용 중 점검을 위함 (From 회의 내용 중)
    - ID 설정 오류가 점등, 통신 상태가 점멸이 되어야하는 것이 아닌지 (From 김용운)
        - 정상적인 상태를 점등, 비정상적인 상태를 점멸로 표시하는 것이 적절할 것 같다는 의견이 많았음 (From 김민수)
        - 정상 운용 상태와 비정상 운용 상태의 Duty를 달리하여 표시하는 것도 사용자 인식에 좋을 것 같음 (From 문부기)
            - 정상 상태 - 90% Duty
            - 비정상 상태 - 10% Duty
    - LED가 점멸하는 것에 대한 사용자의 부정적이 시각이 있음 (서민규)

            - 1안) 과학적 표기법 (Scientific notation) 으로 수정 (AccuraSM 과 유사한 방식)
                - 절대값을 취해서 반올림한 값이 “999.9” 보다 큰 값은 소수점 3자리의 지수(e)로 표시
                    - 예) 12345678 → 1.235e+7
                    - 예) 999.95 → 1.000e+3
                    - 예) 999.91 → 999.9
                - 절대값을 취해서 반올림한 값이 “0.000” 보다 작을 경우
                    - 1-1안) 소수점 3자리의 지수(e)로 표시
                        - 예) 0.123456 → 1.235e-1
                        - 예) 0.0001 → 1.000e-4
                        - 예) 0.0005 → 0.001
                    - 1-2안) “0.000”로 표시
                        - 예) 0.0001 → 0.000
                        - 해당 안 적용 시, 외부 장치가 전력 단위를 kW로 통신하면 0.4W가 “0.0004”로 수신되어 “0.000”으로 표시될 수 있음
            - 2안) 엔지니어링 표기법 (Engineering notation) 으로 수정  (SI 접두어와 호환)
                - 절대값을 취해서 반올림한 값이 “999.9” 보다 큰 값은 Reading 4자리의 지수(e)로 표시하되, 지수(e) 는 3의 배수로 표시
                    - 예) 12345678 → 12.35e+6
                    - 예) 999.95 → 1.000e+3
                    - 예) 999.91 → 999.9
                - 절대값을 취해서 반올림한 값이 “0.000” 보다 작을 경우
                    - 2-1안) Reading 4자리의 지수(e)로 표시하되, 지수(e) 는 3의 배수로 표시
                        - 예) 0.123456 → 123.5e-3
                        - 예) 0.0001 → 100.0e-6
                        - 예) 0.0005 → 0.001
                    - 2-2안) “0.000”로 표시
                        - 예) 0.0001 → 0.000
                        - 해당 안 적용 시, 외부 장치가 전력 단위를 kW로 통신하면 0.4W가 “0.0004”로 수신되어 “0.000”으로 표시될 수 있음
