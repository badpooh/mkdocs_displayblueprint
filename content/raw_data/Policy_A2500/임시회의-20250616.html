안건: [01] [CMS-2P] Load Mode 설정 변경 시, Phase Sequence 설정값 표시 관련 정책

  • 「임시회의-20250612」 [04]에서 3P4W Phase Sequence 설정범위(0:A-B-C-N 디폴트·1:N-A-B-C·2:C-B-A-N·3:N-C-B-A)를 결정. 본 회의는 Load Mode를 3P4W → 3P3W로 변경할 때(설정범위가 0~3 → 0~1로 축소) 범위를 벗어난 값(2,3)의 표시 처리 정책을 논의.
  • 기존 [CMS-2P]의 1P2F·1P2W는 Phase Selection을 같은 번지에서 사용, 1P2F → 1P2W 변경 시 0:Off만 디폴트(1:AN)로 변경되고 1:AN~12:AC는 동일값 유지(참고: 「임시회의-20250612」 image 35 — 3P3W Phase Sequence A-B-C).

👉 Tobe (2안: max-min limited 클리핑)으로 결정 — 범위를 벗어난 값은 해당 모드 범위 내 최대/최소값으로 제한. 내부값(0~3)은 유지하고 표시만 변경(UI에서는 빈 화면, 0/1만 매칭 표시). 기존 1P2F → 1P2W 전환(0:Off → 1:AN, 범위 내 최소값) 방식과의 일관성 유지. 사용자가 임의 변경을 인지하도록 모드 변경 시 Load Wiring 재확인 필요. (각 모드별 Phase Sequence 분리 방안은 복잡성 증가로 미채택. 3P4W 나열 순서는 A-B-C-N·C-B-A-N 우선, N 선행 항목은 후순위 검토.)
