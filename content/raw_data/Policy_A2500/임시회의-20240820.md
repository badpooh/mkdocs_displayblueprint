                - 예) 「SETUP > … > PT Primary L-L Voltage」
                    - 설정범위는 “1~999999”인데, “1000000”을 입력하고 「Enter」버튼 Tap하면, “999999”로 Clipping 되어 Entered됨

            - 설정값이 문자열(”Char”)인 경우: 최대표시범위를 초과한 값 입력 후, 「Enter」버튼 Tap하면, 입력된 문자열의 앞에서 부터 최대설정문자수까지 Clipping 하여 Entered 됨 ← 구현 필요한 사항
                - 예) 「SETUP > … > Device Name」
                    - 최대설정문자수가 “30자”인데, “0123456789012345678901234567890”를 입력하고 「Enter」버튼 Tap하면, “012345678901234567890123456789”로 Clipping 되어 Entered됨
                        - 설정조회화면에는 “01234567890123456789…”로 표시됨 ← 금일 안건 ([설정조회화면에서, 설정값이 문자열(”Char”)이고 가로 길이가 제한된 항목의 최대표시문자수 정책](%EC%9E%84%EC%8B%9C%ED%9A%8C%EC%9D%98-20240820%20f416f87e0c434bd6aa9789baf24a3030.md))의 “Tobe” 안 적용
        - 3안) 최대 입력 자리수를 설정값 입력범위에 따라, 개별 적용
            - 이 때, 최대 입력 자리수는 소수점(”.”)과 마이너스(”-”)를 포함하여 적용
            - 예) 범위가 -999999.9 ~ 999999.9 인경우
                - 최대로 입력 가능한 “-999999.9” 의 9자리까지 입력 가능
                    - 이때, 소수점과 마이너스 없이 “999999999”로 입력 가능함
                        - 단, 「Enter」버튼 Tap하면, 설정조회화면에서는, 최대값인 “999999.9”로 변경되어 표시됨
    - 단, A7300의 Password 관련화면은 예외로 함 (Password는 숫자 4자리로 설정 가능)
        - 「A7300 > RELAY > SETUP > SYSTEM > Security > Password for Local Setup」, 「Password 해제」 팝업: 4자리로 제한
- 👉 Tobe (2안으로 결정)
