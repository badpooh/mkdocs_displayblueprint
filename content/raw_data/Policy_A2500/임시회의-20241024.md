## 안건 3: [A2550VOL] 상단바의 AC/DC 표시 수정 `←「임시회의-20240926」의 설계추가 안건`

>
> - 안건: [[A2550VOL_DC] 화면 설계 완료 공유](%EC%9E%84%EC%8B%9C%ED%9A%8C%EC%9D%98-20240926%200a218a47232e4908a5591521c82fc54b.md)
>     - 👉 결정사항
>         - A2500M 과 A2550VOL 의 상단바 AC/DC 표시를 각각의 설정에 따라 표시
>             - VOL 하위의 CM 모듈의 표시도 설계 수정 필요
- Asis
    - [A2500M] 의 AC/DC 설정에 따른 [VOL] 의 상단바 AC/DC 표시
        - [A2500M] 이 AC 모드로 설정되어 있으면 [VOL] 이 DC 모드로 설정되어 있어도 상단바는 AC 로  표시
        - [VOL] 하위에 연결된 [CM] 모듈도 [A2500M] 의 설정을 따름

        |  | A2500M | VOL | CM (VOL 하위에 연결) |
        | --- | --- | --- | --- |
        | AC/DC 설정 | AC | DC | - |
        | 상단바 표시 | AC | AC | AC (A2500M의 설정을 따름) |

        |  | A2500M | VOL | CM (VOL 하위에 연결) |
        | --- | --- | --- | --- |
        | AC/DC 설정 | DC | AC | - |
        | 상단바 표시 | DC | DC | DC (A2500M의 설정을 따름) |
- Tobe
    - [A2500M] 의 AC/DC 설정과 무관하게 [VOL] 의 AC/DC 설정에 따라, 상단바 AC/DC 표시를 하도록 수정
        - [A2500M] 이 AC 모드로 설정되어 있어도 [VOL] 의 설정이 DC 모드면 [VOL] 은 DC 표시 (VOL의 설정을 따름)
        - [VOL] 하위에 연결된 [CM] 모듈도 [VOL] 의 설정을 따르도록 수정

        |  | A2500M | VOL | CM (VOL 하위에 연결) |
        | --- | --- | --- | --- |
        | AC/DC 설정 | AC | DC | - |
        | 상단바 표시 | AC | DC | DC (VOL의 설정을 따름) |

        |  | A2500M | VOL | CM (VOL 하위에 연결) |
        | --- | --- | --- | --- |
        | AC/DC 설정 | DC | AC | - |
