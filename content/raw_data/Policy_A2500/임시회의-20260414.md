# 임시회의-20260414

## 안건: [01] [A2550IO-DO] 「분기 > SETUP > I/O > DO」 Type 설정범위 "Single Pulse" 추가

- 「A3700N > 임시회의-20251127」 [02]에서 DO Type 설정범위 추가(0:Latch·1:Pulse·2:Energy Pulse·3:Single Pulse, CB 제어 명령 시 Type 자동 "3:Single Pulse"). A2550IO-DO에 Single Pulse 추가 검토.

> [이미지 image.png] [참고/메일 문부기 2026-04-06] "A2500 일정 공유 및 추가 기능정의" — A2500 릴리즈 일정(LG U+ App 1.16.0 4/30 목표·A2550 VOC App 1.15.1 최소기능·VOC 최종 App 1.16.1). A2550 IO 추가기능: LG U+ IO에 DO 포함, Latch/단발성(single) 펄스 사용, 현재 Latch·Periodic Pulse만 있고 Single Pulse 없음 → DO 모드에 Single Pulse 추가. Single Pulse 동작: Command "1"만 받고 "0" 미동작, "1" 수신 시 즉시 "0" 복귀·pulse 출력 시작, On time+Off time 한 주기 완성까지 추가 command 무시.

### Tobe (DO Type 설정범위 추가)

> [이미지 image 13.png] [Tobe/분기 SETUP DO Single Pulse, ID32 IO-DO1] SETUP > I/O > Digital Output 1 — DO.1: Polarity Normal·Type & Hold Mode Single Pulse·Pulse Period 0.20·Pulse On Time 0.10·Counter Reset·Control(Hold Time 비활성).

> [이미지 image 14.png] [Tobe/Type 설정팝업 Single Pulse, ID32 IO-DO1] "Type" 팝업 — Latch·Pulse·Single Pulse(선택), Hold Mode 비활성.

- Tobe: 1안) Energy Pulse 미고려(0:Latch·1:Pulse·2:Single Pulse) / 2안) Energy Pulse 추후 추가 고려(0:Latch·1:Pulse·3:Single Pulse, 통신 "2" 입력 시 Command 무시). Control 동작: "3:On"만 받고 "2:Off" 미동작. Single Pulse로 설정된 DO Notification은 이벤트 발생 시 DO 출력 동작. A2550IO-DO2·DIO 해당, 자체 이벤트 연동 DO 포함 모듈(CMDR+RM·TEMP·DCM·SAL 등)에는 미적용.

**👉 결정사항** — DO Type 설정범위 추가: 2안으로 결정(단 Energy Pulse는 추가 안 함). 설정범위: 0:Latch(디폴트)·1:Pulse·2:Reserved·3:Single Pulse.
