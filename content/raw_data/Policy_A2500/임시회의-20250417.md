# 임시회의-20250417

## 안건: [01] [A2550IO-RTD1]/[RTD2] 계측 범위에 따른 계측값 표시

- RTD(측온저항체) 계측값이 설정 계측 범위를 벗어날 때의 표시 방안 논의.

**👉 Tobe 안으로 결정** — 계측 범위 미만/초과 시에는 미연결과 동일하게 OPEN(또는 하이픈)으로 표시, 범위 내일 때만 정상 계측값 표시. 표시 범위: PT100 10.00~400.00Ω → 10.0~400.0Ω, 온도환산 -218.7~882.6℃, PT1000 100~4000Ω.

## 안건: [02] [CMS-2P] Load Mode가 3P3W인 경우 Residual/전류 영상분 표시 여부

- [CMS-2P] 모듈의 Load Mode를 3P3W로 설정한 경우, 영상분 성분 관련 계측항목(Residual Current, Curr. Symm. Zero-Sequence, Curr. Unbal. % Zero-Sequence)의 표시 여부 논의. 3P3W(3상 3선식)에서는 영상분이 물리적으로 의미가 없으므로 표시 정리 필요.

> [이미지 image.png] [참고/SETUP Load Mode, ID01 CMS-2P-60A] SETUP > MEASUREMENT > Load Mode — Mode 「3P3W」 설정 상태.

> [이미지 image 1.png] [Asis/HOME CURRENT Residual, ID01] HOME > CURRENT > Residual — Residual Current Fund. 0.000A·RMS 0.000A(3P3W에서 영상분 0 고정 표시).

> [이미지 image 2.png] [Asis/ANALYSIS Curr.Symm., ID01] HOME > ANALYSIS > Curr. Symm. — Curr. Symm. Component: Positive I1 29.94A·Negative I2 0.034A·Zero I0 0.000A(빨간 박스, 3P3W에서 영상분 0 고정).
