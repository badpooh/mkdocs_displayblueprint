# 임시회의-20240530

## 안건: AC/DC 모드에 따른 모듈별 화면 표시 정책

- 「임시회의-20240725」에서 설계추가. Asis: A2500 "AC/DC 모드 설정"과 연결된 A2550 모듈 타입(AC/DC)이 다른 경우 분기 화면을 모두 표시 → 사용자 혼란. 미설계 모듈 화면 설계/구현/테스트 필요(AC모드 [CMDR+RM], DC모드 [CMZ-1P/3P]·[CM-3P]·[CMS+ZM]·[CMM-3P]·[CMMZ-3P]).

> [이미지 Untitled 1.png] [Tobe/HOME AC/DC Mismatch, MAIN] HOME — 상단바 경고 아이콘 "AC/DC Mismatch", VOLTAGE L-L 380.0V·Freq 60Hz는 표시되나 CURRENT/POWER/ENERGY 계측값 "-" 표시.

- Tobe(상단바·ChainView·StatusView·EVENT): 메인은 "AC/DC Mismatch" 발생 시 모듈 선택창 2초 토글(기존 "MAIN ID 00"·ID 배경색 #E6E6E6 + 경고 아이콘·"AC/DC Mismatch"), 분기도 동일 토글. (상단바·ChainView·EVENT 화면 예시는 「임시회의-20260120」 자료 참고.)
