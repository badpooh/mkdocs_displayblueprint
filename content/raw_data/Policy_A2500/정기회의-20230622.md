# 정기회의-20230622

## 안건: [A2550IO] 모듈 ID 범위 (`정기회의-20230629` 결정)

- PowerDX가 "50"을 Special ID로 사용(A2300·A2500 ID). Tobe: 물리 ID(로터리스위치 1~15, "0"은 공장초기 예약), "모듈 ID" = 물리 ID + 50 → 범위 51~65, 255(디폴트, 물리 ID "0").

**⏭️ 범위를 61~75로 확장(Reserved 확보) 검토** — 장치 "Internal ID" 범위 확인(임베디드SW파트) 후 결정.
