# 임시회의-20260604

## 안건: [01] [A2550IRT] 모듈 설계 방향 공유

- 상세모델명 A2550IRT-3P(특수모듈). MCCB에 체결된 버스바 선로의 온도를 적외선 측정 방식으로 계측(버스바 선로 3개 측정, 차후 4개까지 확장 가능). 설계 완료가 아닌 설계 방향 공유. (A2550 IRT 요구사항 정리서.docx 별도 첨부.)
- 연결 방식: ① A2550CMS-2P에 서브모듈로 연결(Module ID 미할당, 기존 ZCT 연결 단자 대신 IRT 연결) / ② A2500M에 직접 연결(Module ID 할당).

> [이미지 image.png] [참고/메일 문부기 2026-06-01] "A2550 IRT 모델 개발방향 검토사항" — ① CMS-2P-IRT 모델 개발 시 메모리 문제(CMS-2P는 K10 개발, RAM 여유 약 5.5KB로 부족, IRT 연결기능 추가 시 모듈 통신·관리 구현으로 메모리 추가 소모). ② CMS-2P-IRT 추가 시 상위 CPU/UI 변경(외부 데이터구조·가공구조 개선, 결선모드별 IRT 처리 결정, 온도 화면구조·Chain View·Navigation Bar 변경). ③ A2550IRT를 A2500에 직접 연결 시 CMS-2P 변경 없음·A2500M/D에서 IRT 추가·ChainView ID 변경기능에 IRT 포함(단 grouping 지원 시 변경 필요). IRT만 개발 시 7월말, CMS-2P 연동까지는 8월말 예상.

- 구현 우선순위: ① 독립 모듈 직접 연결(1차 타겟) → ② CMS-2P 종속 연결(2차) → ③ Pairing(3차). 7월 중 일정, 고객 피드백 받아 페어링·이벤트 우선순위 조정 예정. (용어: "직결"=1대1 연결, "데이지체인 연결"=상위 모듈 연결, 혼동 방지 위해 구분 사용.)
