### 「상단바 > ◀/▶」 모듈 이동 순서 및 분기 화면 메뉴 구성

> [이미지 Untitled 13.png] [참고/모듈 이동 순서, MAIN ID00] HOME Dashboard(VOLTAGE Avg 380.7V·Freq 60Hz, CURRENT 29.96A·Leakage 2mA, POWER 17.127kW PF Lag 0.865, ENERGY Net 474,457kWh, SYSTEM ROOTECH) — ID 이동은 오름차순(같은 모델은 화면 Index 유지, 다른 모델은 모델별 최근 화면 Index, 최근 Index 없으면 대시보드, 휘발 메모리·전원 Off 시 사라짐). CM-3P 설정(Load Mode 3-phase/1-phase, 외부ZCT 사용 Enable/Disable)에 따라 각각 모델 취급, CMS의 ZM 결합 유무에 따라 각각 모델 취급(MCCB Status 설정은 무관).

- 분기(A2550VOL) "AC" 모드 메뉴(DC 모드는 차후): HOME은 A2500M과 유사하나 「SYSTEM」 부메뉴 다름(Local Time·A2500D 없음, 자체 "A2550VOL"·하위 "A2550CMZ"). SETUP은 「NETWORK·DEVICE·CONTROL」 주메뉴 없음, MEASUREMENT는 Voltage만(Demand·Power·AC/DC 없음), EVENT는 Dip·Swell·PQ Curve·Volt. Connection(Transient Volt.는 차후, DO Notification은 A2500M 구현 후 추가), SYSTEM은 Description만. Demand·Power·Module ID·MCCB Status·Data Reset·Security 등은 「A2500M > SETUP」 설정을 따름. EVENT 모드는 「SYSTEM」 주메뉴 없음, MEASUREMENT(All/Voltage/Current/Power/Leakage/Etc.)·POWER QUALITY(All/Dip/Swell/SEMI F47/IEC 61000/ITIC, Transient Volt.는 차후). 통신은 "Event RMS Trend"만 제공("Event Waveform Data" X).

**👉 결정사항** — 모듈명 표시: VOL에 CM 정상연결 시 (1안)으로 결정. CM Disconnected 시 ID 미표시·"--" 표시(VOL은 "Main CM" 설정이 없어 ID 표시 불가).
