정기회의-20231012

안건: [CMMZ]·[CMS+ZM] 「분기 > SETUP」에 「I/O」 주메뉴 추가에 따른 「SETUP Dashboard」 수정

  • 「정기회의-20230907」·「정기회의-20230921」 결정([CMMZ]·[CMS+ZM]의 "MCCB Status"를 「분기 > SETUP > MEASUREMENT」에서 「I/O > DI」 주메뉴로 이동)에 따라, SETUP Dashboard(설정 첫 화면)도 「I/O」 영역이 추가되도록 수정 검토.

👉 「분기 SETUP Dashboard」에 「DI」 영역 추가로 결정 ([ZM] 결합 모듈만 표시, 단독 모듈은 미표시)

안건: [A2550SAL] 「ANALYSIS > Harmonics」 Leakage 단위 콤보박스 수정

  • Harmonics 화면의 Leakage 탭에서 단위 콤보박스를 검토. Asis는 [mA] 기준이나, 단위 선택 옵션 추가 검토.

👉 Harmonics Leakage 단위 콤보박스에 [A] 옵션 추가로 결정

안건: [A2550SAL] 상세모델명 "A2550SAL-3P-DO" 재변경

  • 「정기회의-20230914」에서 검토한 SAL 상세모델명(해당 회차 자료 Untitled 32~34 참고)을 "A2550SAL-3P-DO"로 재변경하는 안 논의.

👉 상세모델명 "A2550SAL-3P-DO"로 재변경

안건: [A2550SAL] 「SETUP > EVENT > Leakage」 화면 — 누설전류 판정기준 의견 취합

  • SAL 누설전류 판정기준(피뢰기 누설전류) 자료를 취합하여 EVENT > Leakage 항목 설계에 반영. 단위·항목 구성을 차주 결정 예정.
  • KESCO 환경기준(23kV 요주의/주의 판정 표)은 「정기회의-20231005」 자료(Untitled 6) 참고.

👉 결정사항 — EVENT > Leakage 항목은 "It 하한, It 상한, I3h 상한, It 과소" 4개 항목만 두기로. 히스테리시스·Time Delay는 삭제, Trigger·3rd Harmonic Unit은 유지.

안건: [A2550MVM] 「HOME > Dashboard」 Voltage 표시 방식

  • MVM(다채널 전압 계측) Dashboard에 Voltage를 어떻게 요약 표시할지 논의(Average/Maximum/Minimum 등).

👉 결정사항 — Dashboard Voltage는 Average/Maximum/Minimum 표시, Max/Min은 해당 채널번호도 함께 표시.

안건: [A2550M] (Sub-Module for Multi-Power) 모듈 설계 1차 공유

  • 기본 컨셉: [A2550M]은 다중 전원 계측용 Sub-Module. AC/DC 설정 가능. DC 계측 가능성 검토(변영복: 적용 가능성을 열어두라는 부탁을 받음 → DC 모드는 CMD·CMDR+RM에 연결 가능한 걸로 정리).

👉 결정사항 — AC/DC 설정 가능(DC는 Subframe Type 삭제), StatusView는 [M+CMS+ZM]에서 미표시, ChainView 표기(1안/2안)는 추가 검토. ([M] 결합형 상세 화면은 「임시회의-20251030」 자료 참고.)
