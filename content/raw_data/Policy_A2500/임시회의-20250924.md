# 임시회의-20250924

## 안건: [01] [CMS-2P_3P4W] 「분기 > SETUP > MEASUREMENT > Load Mode」 "Pairing Module ID" 설정변경 정책

- Pairing Module ID 설정 로직 및 예외 처리(장치 기준). 설정 거부 조건: ① Pairing Main에서 자신의 ID를 Pairing Module ID로 설정(예: CMS-2P ID00에서 "0"), ② 이미 Pairing된 CMS-2P를 다른 CMS-2P에서 Pairing Module ID로 설정(예: ID00+ID01 상태에서 ID02가 "0"/"1" 설정 → ID02 비정상·ID00+ID01 정상).
- 장치에서 Pairing Module ID가 "255"로 적용되는 경우: ID00+ID01 상태에서 ① ID00의 Load Mode를 "3P3W"로 변경(Pairing 해제·Pairing Module ID "1" Disabled) → ② ID02에서 Pairing Module ID "1" 설정(ID02+ID01) → ③ ID00에서 Pairing Module ID "1" 설정 시 변경 불인정하고 "255"로 적용.
- AccuraSM 동작 방식: Pairing Module ID 설정변경 시 ① 255(디폴트)로 설정 → ② 목표 ID로 변경(예: 6→7은 6→255→7, 단 화면은 255 미표시하고 6→7). AccuraSM은 여러 설정을 동시 변경하는 경우가 많아 이 로직이 추가되어 장치와 상이.

CMS-2P-400A/60A·CMZ-3P-60A 모듈별 CT Ratio·Rated Current·Leakage Current 설정 포함.

- Tobe: AccuraSM 동작 방식의 Asis 유지 여부 결정 필요.

**👉 결정사항** — AccuraSM 동작 방식: 기존에 선점되어 있는 Pairing Module ID의 설정값만 255를 거쳐서 변경. Pairing Module ID 변경 관련 이벤트로그에는 ID 255 변경 이력이 기록됨.
