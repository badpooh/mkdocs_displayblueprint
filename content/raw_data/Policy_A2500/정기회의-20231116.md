## 안건: [A2550TEMP] THSEN 기능 추가에 따른 화면 표시

> [이미지 Untitled 10.png] [요청/메일 문부기 2023-11-08(윤우진 전달)] "TEMP 제품 THSEN 기능 추가에 따른 화면 표시" — A2350/2550 TEMP에서 Diff Event Setup 시 TSEN 지정, THSEN 릴리즈로 "Tsen"만 표시가 적절한지 지적. A Select: Out→Tsen.1~6→Tsen.High 로테이션. 기존 "Tsen.(n)" → 수정 "T(H)sen.(n)"(EVENT SETUP DIFF EVENT 화면 비교). 현재 미장착 TSEN도 1~6 모두 Tsen 표시.

> [이미지 Untitled 11.png] [답변/메일 김민수 2023-11-09] A2550TEMP 화면 설계서 작성 시 검토·회의 예정(VOL 등 신규 모듈 설계 중이라 시간 더 필요). 의견: (1) TSEN·THSEN 아우르는 단어 "SENSOR"로 잠정 합의(with 조국춘), (2) 모듈명 지칭 시 "TSEN", 센서가 제목 아닌 경우 첫 문자 대문자 "Sensor", (3) TSEN은 DI/DO처럼 고정 채널 아니므로 현재 달려있는 센서만 표시.

**👉 결정사항** — (1) "TSEN"·"THSEN" 아우르는 용어는 "SENSOR" 사용, (2) 모듈명 지칭 시 "TSEN", 센서가 제목 아닌 경우 "Sensor"(채널과 함께 표시 시 "Sensor.1"), (3) SETUP에서는 현재 미장착 센서도 로테이션 포함·미리 설정 가능, HOME에서는 미장착 센서 미표시.
