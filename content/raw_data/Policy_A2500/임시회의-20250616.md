# 임시회의-20250616

## 안건: [01] [CMS-2P] Load Mode 설정 변경 시, Phase Sequence 설정값 표시 관련 정책

- 「임시회의-20250612」 [04]에서 3P4W Phase Sequence 설정범위(0:A-B-C-N 디폴트·1:N-A-B-C·2:C-B-A-N·3:N-C-B-A)를 결정. 본 회의는 Load Mode를 3P4W → 3P3W로 변경할 때(설정범위가 0~3 → 0~1로 축소) 범위를 벗어난 값(2,3)의 표시 처리 정책을 논의.
- 기존 [CMS-2P]의 1P2F·1P2W는 Phase Selection을 같은 번지에서 사용, 1P2F → 1P2W 변경 시 0:Off만 디폴트(1:AN)로 변경되고 1:AN~12:AC는 동일값 유지(참고: 「임시회의-20250612」 image 35 — 3P3W Phase Sequence A-B-C).

> [이미지 image.png] [참고/Accura 2500 Modbus Map Phase Selection] 7 Phase sequence(UINT16·RW·Default 0)·결선에 따른 상 표시 순서, CMS-2P 3P3W에서 유효(0:ABC/1:CBA). 8 Feeder 1: Phase selection(범위 0~12·Default 1:AN, CMS-2P 1P2F·1P2W에서 유효, 0:OFF·1:AN~6:CA·7:NA~12:AC, 빨간 박스)·9 Feeder 2: Phase selection(Default 2:BN).

> [이미지 image 1.png] [참고/분기 SETUP Load Wiring Phase Selection, ID01 CMS-2P-60A] SETUP > MEASUREMENT > Load Wiring — Feeder1/Feeder2 탭, Load Direction* >>>·Phase Selection AN. 항목범위: Load Direction(0:>>>/1:<<<, 외부ZCT 연결 시 영향 없음)·Phase Sequence(0:A-B-C/1:C-B-A, Mode 3P3W에서 표시)·[Feeder1] Phase Selection(0:Off~12:AC·Default 1:AN, Mode 2:1P2W or 3:1P2F에서 표시, 1개 피더 제외 나머지 모두 Off 시 나머지 1개 Off 불가, 1P2W에서 0:Off 항목 숨김)·[Feeder2] Same as Feeder1(Default 2:BN).

- Tobe: Load Mode를 3P4W → 3P3W로 변경 시 Phase Sequence 표시 — 1안) 디폴트값으로 표시(2,3 → 0:A-B-C), 2안) 설정범위 내 최대값으로 표시(클리핑, 2,3 → 1:C-B-A). 두 안 모두 0:A-B-C-N → 0:A-B-C, 1:N-A-B-C → 1:C-B-A.

**👉 Tobe (2안: max-min limited 클리핑)으로 결정** — 범위를 벗어난 값은 해당 모드 범위 내 최대/최소값으로 제한. 내부값(0~3)은 유지하고 표시만 변경(UI에서는 빈 화면, 0/1만 매칭 표시). 기존 1P2F → 1P2W 전환(0:Off → 1:AN, 범위 내 최소값) 방식과의 일관성 유지. 사용자가 임의 변경을 인지하도록 모드 변경 시 Load Wiring 재확인 필요. (각 모드별 Phase Sequence 분리 방안은 복잡성 증가로 미채택. 3P4W 나열 순서는 A-B-C-N·C-B-A-N 우선, N 선행 항목은 후순위 검토.)
