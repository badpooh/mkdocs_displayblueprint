# 정기회의-20230420

## 안건: 「SETUP > … > 숫자패드형 수정팝업」 숫자패드 형태 통일

- 「정기회의-20230406」 재논의. 범위 표시 기호는 「정기회의-20230406」 결정(en dash "–") 따름. A7300 회의 디자인시안 기준으로 "±" 버튼을 **"–"(en dash)**로 수정.

> [이미지 Untitled.png] [Tobe/숫자패드 3종] NUMBER PAD — Type 1(기본, 우측 X·Clear, 하단 0·.·–), Type 2(상단선택용, 4열, .·–), Type 3(Password, 2행 1~0). "±"→"–"로 수정, 버튼 표시순서·Delete 모양 변경 주의.

> [이미지 Untitled 1.png] [Tobe/적용 예시] "Min. Meas. Secondary Vtg." 팝업 — Range 0–50, 입력값 5, Type 1 숫자패드("." "–" Disable, 정수·양수 범위라).

- 동작 규칙: 음수 입력 시 "–" 먼저 입력 후 숫자. "." 버튼 — 팝업 열릴 때/입력필드 빈칸일 때 Range가 소수점 포함하면 Enable(정수 범위는 Disable 유지), 입력필드에 "."이 이미 있으면 Disable. "–" 버튼 — 팝업 열릴 때/빈칸일 때 Range가 음수 포함하면 Enable(양수 범위는 Disable 유지), 입력필드가 빈칸 아니면 Disable.

## 안건: 「상단바 > 모듈선택창 ◀/▶」 모듈 이동 순서

- 「정기회의-20230407」 재논의. 2023/04/12 영업의견(안창모) 청취 — 영업부는 예전부터 "ID 오름차순" 이동 요청, 0407 결정에 동의.
- Tobe: **ID 오름차순 이동.** 같은 모델이면 화면 Index 유지, 다른 모델이면 모델별·모드별 최근 화면 Index로 이동(없으면 현재 모드 대시보드, 대시보드 없는 모델은 모드별 첫 메뉴 첫 화면). 최근 화면 Index는 휘발 메모리 저장(전원 Off 시 사라짐).
- 모델 구분: CM3P는 "Load Mode"(3-phase/1-phase)·"외부ZCT 사용"(Enable/Disable)에 따라 각각 별도 모델 취급. CMS는 ZM 결합 유무로 별도 모델 취급(분기 MCCB Status 설정은 무관). IO는 10개로 구분.

## 안건: 「ChainView」 모듈 연결/탈락 표시 방안

- 「정기회의-20230407」 재논의. 영업의견 — ChainView는 현장 사용자보다 영업·대리점이 설치·유지보수에 주로 사용, 0407 결정(달라진 부분만 간단히 표시) 동의, 추가된 모듈도 표시 요청.

Reset 방식: Continuous(사용자 Reset)/Self-resetting(이벤트 종료 or 사용자)/Definite Time(설정시간 경과 or 사용자). 이벤트별 알림 — PQ·전압연결·계측·모듈탈락 "시작"은 O, "종료"·모듈추가·링해제·연결저장·통신·설정변경은 X.
