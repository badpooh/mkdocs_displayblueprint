# 임시회의-20260507

## 안건: [01] 2038 Time Overflow 관련 장치별 수정

- 2038년 UNIX 시간 overflow 대응. 4Byte Signed Type은 2037/12/31까지만 표현 가능 → overflow 발생. (Time overflow 테스트·대응 pdf, A3500F/E 부팅불량 시료 분석 pdf, A3700N RTC 이상 시료 분석 docx 별도 첨부.)

> [이미지 image.png] [참고/메일 김민제 2026-05-21] "Accura 제품군 Time overflow 대응 관련 사항 공유" — 2038년 UNIX 시간 overflow 예정, 장치 운용 수명 10년 기준 2028년까지 대응책 적용 필요. Time overflow 개요·대응책·수정사항 테스트·상위 시스템 지원 정리, A3500F/E로 테스트 버전 준비 예정.

> [이미지 image 1.png] [참고/메일 김민제 2026-01-07] "A3500F/E 부팅 불량 시료 분석 결과" — 분석 대상 장치 접수번호 AS25B11002·AS25C11002.

> [이미지 image 2.png] [참고/메일 문부기 2026-01-07] "FW: A3500F/E 부팅 불량 시료 분석 결과" — A3500F 불량시료 분석, RTC상에 2043년대 시간 기록·시스템 시간이 2038 문제 겪으며 나타나는 증상.

> [이미지 image 3.png] [참고/메일 김민제 2026-01-13] "A3500F/E 부팅 불량 시료 분석 보강" — 분석 대상 접수번호 AS25B11002·AS25C11002, 1차 분석 결과도 첨부.

> [이미지 image 4.png] [참고/메일 문부기 2026-01-13] "FW: A3500F/E 부팅 불량 시료 분석 보강" — A3500F RTC 값 깨짐으로 인한 부팅불량 추가분석. 제품 전원 꺼진 채 오래 방치되어 RTC 배터리 경우 추측되나 재현시험 미재현, SW 보완 릴리즈 진행 필요.

> [이미지 image 5.png] [참고/메일 김민제 2026-04-24] "A3700N RTC 이상 현상 시료 분석 보고서" — 접수번호 AS26421005.

> [이미지 image 6.png] [참고/메일 문부기 2026-04-28] "FW: A3700N RTC 이상 현상 시료 분석 보고서" — A3700N RTC 이상으로 인한 부팅불량 현상 분석자료.

- 장치별 현황(참고): A7300·A2700MHN·A2700M 미반영(4Byte Signed, 1970~2037, A7300·A2700MHN은 8Byte 적용 가능·A2700M은 HW 제약으로 8Byte 불가).
- Asis: A3500·A2300 장치 반영/SM 미반영(4Byte Unsigned, 1970~2106, 범위 초과 시 Time Overflow). A3700N 미반영(4Byte Signed, 1970~2037). A2500 미반영(4Byte Signed, 2000~2037).
- Tobe: Format을 "4Byte Unsigned Type"으로 수정, Local Time 설정범위를 "2000/01/01 00:00:00 ~ 2069/12/31 23:59:59"로 수정(장치·AccuraSM·Modbus Poll 동일). A3500·A2300·A3700N·A2500 모두 장치/SM 반영, 동작범위 초과 시 2106/02/07 06:28:15까지 동작.

**👉 Tobe 안으로 결정** — 임베디드 장치(A2500·A3700N 등)에 4Byte Unsigned 포맷·설정범위(2000~2069) 적용. A7300·A2700은 산업전자 팀과 별도 회의. 장치·SM·Modbus 모두 동일 적용. (RTC 최대 2099년 지원이나 장치 수명 고려 2069년으로 설정, 추후 플랫폼 8Byte 전환 예정.)
