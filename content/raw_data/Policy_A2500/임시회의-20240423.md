            | --- | --- |
            | 0 | 255 |
            | 1 ~ 9 | 31 ~ 39 |
            | A(10) ~ F(15) | 255 |

## 안건 5: 제품공통정책으로 추가 필요한 안건
- “Demo Test Mode”에서, Demo 파형을 “Reference Voltage”를 기준으로 생성하도록 수정

    > 참고: 장치별 “Demo Test Mode”와 “Reference Voltage” 설정 위치
    >
    > - A2500
    >     - 「메인 > SETUP > CONTROL > Test Mode」
    >     - 「메인 > SETUP > MEASUREMENT > Voltage > Reference Primary Voltage [V]」
    > - A3700N
    >     - 「SETUP > CONTROL > Test Mode」
    >     - 「SETUP > MEASUREMENT > Voltage > Reference Voltage [V]」
    > - A7300_METER
    >     - 「SETUP > CONTROL > Test Mode」
    >     - 「SETUP > MEASUREMENT > Voltage > Primary Reference Voltage [V]」
    - Asis
        - [A2500], [A3700N], [A7300_METER]에서, Demo를 상전압 220V 기준(고정값)으로 파형 생성하고 있음
            - 그래서, [A7300_METER]과 같이 Default 설정(190V)이 낮은 경우, Test Mode 사용시에 Dip 발생되지 않음
    - Tobe
        - Demo 파형을 “Reference Voltage”를 기준으로 생성
            - 파형 생성시, “L-L”과 “L-N” 설정도 확인 필요
    - 👉 결정사항
        - “Demo Mode”내에서, 항목값이 변경되는 시점의 “Reference Voltage”으로 파형 생성
            - 예) “Dip Short”에서 “Dip Long”으로 변경될 때, 그 시점의 “Reference Voltage”로 파형 생성
            - 예) “Off”에서 “Balance”로 변경될 때, 그 시점의 “Reference Voltage”로 파형 생성
            - 예) “Dip Short”이 동작하는 중간에, “Reference Voltage”을 변경해도 파형 생성기준이 변경되지 않음
        - “L-L”로 설정되어 있으면, “L-N” 값으로 변환하여 장치에 적용됨
            - 예) “L-L”, 380V로 설정되어 있으면, 장치에서는 “L-N”, 220V로 Demo 파형 생성
- 장치화면 내의 버튼 동작 시점 통일 (Pressed/Released)
    - Asis: 제품별로 버튼 동작 시점이 다름
        - Pressed/Released를 구분한 방법: 손가락을 한 지점에서 누르고, Drag 하여 다른 지점에서 손을 떼어 동작여부 확인
            - Pressed: 한 지점을 누를 때, 기능 동작 함
            - Released: Drag 하여 다른 지점에서 손을 떼면, 기능 동작 안함
        - 3개 제품 모두, Released에서 동작을 기본으로 하고 있는나, 같은 UI 요소에서 다르게 동작하고 있음


            |  | Pressed | Released | 비고 |

            | A2500 | 모든 Dashboard,
            EVENT의 표 | 주/부메뉴,
            HOME의 버튼,
            EVENT의 버튼,
            SETUP의 설정항목 | - 버튼 Pressed (Highlight) 상태 표시 함
            - 팝업의 버튼은 Pressed와 Released가 혼용됨 |
            | A3700 | 모든 Dashboard,
            EVENT의 표,
            SETUP의 설정항목 | 주/부메뉴,
            HOME의 버튼,
            EVENT의 버튼, | - 버튼 Pressed (Highlight) 상태 표시 함
            - 팝업의 버튼은 Pressed와 Released가 혼용됨 |
            | A7300 | 팝업의 일부 버튼 | 주/부메뉴,
            모든 Dashboard,
            팝업의 일부 버튼 | - 버튼 Pressed (Highlight) 상태 표시 안함
            - 팝업의 버튼은 Pressed와 Released가 혼용됨 |
            - A2500, A3700N의 “표”는 스크롤 지원을 위해, Pressed로 동작 필요
            - A7300은 표에서 스크롤 안됨 (세로스크롤에서 상/하 버튼만 동작)
            - 3개 제품 모두에서, 팝업의 버튼은 Pressed와 Released 혼용되고 있는데, 모두 제각각으로 동작됨
