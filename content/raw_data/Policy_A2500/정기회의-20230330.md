# 정기회의-20230330

## 안건: 계측화면과 Dashboard 화면 값 표시 통일

- 「정기회의-20230316」 TFT_LCD 숫자값·단위 표시 정책에서 Dashboard 표시값 정책이 누락되어 보완. Dashboard 값과 계측화면 값이 같도록 함.

> [이미지 Untitled.png] [참고/추가규칙] 전력량은 "[Wh]"/"[kWh]" 정수, ZCT는 "[mA]" 정수(0~2,000mA), 3상 단위가 다르면 3상 중 Max값의 단위·소수점 자리수 따름(Tot·Avg 동일).

> [이미지 Untitled 1.png] [참고/추가규칙 확장] 보호계전용 ZCT(A7300)는 2차측 0~10mA(정격 1.5mA)로 mA 소수점 표시·[mA] 4자리, 대표값 "Tot"/"Avg"만 표시되는 경우도 동일(예: HOME > Dashboard), CMS_1P3F·CM_3P "1-phase mode"는 각 피더(상)별 독립 단위·소수점.

## 안건: 「SETUP」 확인팝업 문구 수정

- 매뉴얼 확인(2023/03/17): "system"은 host system을 지칭하나 TFT_LCD에서 「SYSTEM」 주메뉴로 두루 쓰여 "system" 사용으로 확정.
- 대상: 「SETUP > DEVICE > Module ID」·「SETUP > CONTROL > Reset」 전 항목(DO 출력 테스트는 확인팝업 없음). 정책: **장치 자체 입장만 고려**(상위시스템 문제는 미고려) — 적산값 Reset도 장치 입장에선 Caution 수준(Undo 불가).

## 안건: 「SETUP > … > Setup Exit Timeout [sec]」 항목 추가

- 기술영업부(안창모, 2023/03/17) 요청: A3700에 있던 "SETUP 모드에서 일정시간 입력 없으면 HOME 자동전환" 기능이 A3700N·A2500에서 빠짐 → 추가. (EVENT 모드는 유지.)

> [이미지 Untitled 14.png] [참고/이메일] "[스마트피앤티팀] LCD제품의 Setup Exit Timeout 기능에 대한 의견 요청"(2023/03/21).

> [이미지 Untitled 15.png] [참고/이메일 2023-03-24, 안창모] LCD 크기 무관하게 모든 제품에 "설정모드종료" 필요(고객이 설정 조작에 민감, TFT는 버튼 한 번에 진입돼 더 불안). 신규 제품 바로 적용, 기존 제품도 반영.

- Tobe(결정): 변경/조회 모드 무관하게 Setup Exit Timeout 후 HOME 전환, EVENT는 유지. 항목명 **「Setup Exit Timeout [sec]」**(A2550 "LCD Setup Exit Timeout"에서 "LCD" 삭제). 범위·디폴트 — 메인장치(LCD 大) 60~3600(디폴트 600초), 모듈장치(LCD 小) 5~60(디폴트 30초). A2350CT는 HW 사양으로 고정(10초·30초). LCD화면·통신맵 공개. 신제품 즉시 적용, 기존 제품은 「제품정책결정사항관리」로 관리.
