## 안건: [02] A2500의 재부팅 기능 추가

- 「임시회의-20240912(긴급)」 추가안건 설계추가. 서민규 요청(현장에서 차단기 동작 없이 장치만 재부팅 필요 — 통신 리셋·화면 멈춤 해소). A2300·A3700OLD·A3700N은 물리버튼 4개 약 10초 Press로 재부팅(매뉴얼에 미기재), 통신 재부팅 불가·하위 모듈은 재부팅 안 함. (장치별 재부팅 영상 A2300.mp4·A3700OLD.mp4·A3700N.mp4 — 동영상 자료.)
- Tobe: A2500 재부팅 기능 추가 — 물리버튼 4개(HOME·SETUP·EVENT·BACK) 약 10초 Press로 동작(A3700N과 유사, POWER 제외 LED 소등·LCD Backlight Off 후 재부팅), 통신 재부팅 불가, 재부팅 시 A2550은 재부팅 안 함.

**👉 결정사항** — 재부팅 기능 추가하되 적용 시기 추후 재검토. 회의내용: 버튼 통한 Reboot는 Kernel 단 기능, 디스플레이 분리형 제품은 Main 제품 Reboot 가능 여부 확인 필요.
