# Policy-23-0031 · 「EVENT > DO Reset」 버튼 표시

## 결정 사항

- 「DO Reset」·「종 아이콘」 버튼 **위치 고정** — EVENT 모드에서 새 이벤트 발생 시 함께 표시(단 DO Reset은 표시 조건 만족 시만). 3개 버튼 위치 고정하고 표시/숨김 처리.

> [이미지 Untitled 115.png] [Tobe/A2500] EVENT 화면(Measurement 45/100). 상단 우측에 **AC·DO Reset·종 아이콘(005)** 버튼이 위치 고정 표시.

- **「DO Reset」 버튼 표시 조건** (AND): ①DO 채널 포함 모듈 연결됨 ②DO 출력 설정 모두 Enable(Trigger·DO Notification) ③이벤트 알림으로 DO 출력 동작 중
- 「DO Reset」 Tap 시 글자색 변경되고 확인팝업 열림

> [이미지 Untitled 116.png] [Tobe/A2500] DO Reset 확인팝업: "Performing this operation requires caution. Do you want to continue?" (Confirm/Cancel).

- 확인팝업 「Confirm」 시 DO 출력 Reset되고 「DO Reset」 버튼 숨김

> [이미지 Untitled 117.png] [Tobe/A2500] DO Reset 후 화면 — DO Reset 버튼 숨겨지고 종 아이콘만 표시.

## 결정 배경

- Dip 등 이벤트 설정 시 "DO Notification" 설정 가능, 발생 시 「EVENT > DO Reset」으로 Reset. A7300은 이 버튼 없음(메터부 기능 없음, 계전부는 「Fault Reset」 물리버튼). Asis: DO 모듈·설정 여부와 무관하게 버튼 항상 표시.
