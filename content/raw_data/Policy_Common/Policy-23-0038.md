# Policy-23-0038 · SETUP 항목 Disabled/Hidden 자동변경 정책

## 결정 사항

- 주메뉴/부메뉴 구성이 **변경되는** 설정항목: 「SETUP > Dashboard」로 화면 자동전환 (예: 「MEASUREMENT > AC/DC > Mode」)
- 주메뉴/부메뉴 구성이 **유지되는** 설정항목:
  - 같은 화면에서 영향받으면 **Enabled/Disabled/ReadOnly** 처리
  - 다른 화면에서 영향받으면 **Shown/Hidden** 처리

> [이미지 Untitled 297.png] [같은 화면/ReadOnly] SETUP > MEASUREMENT > Voltage. Wiring "1P2W" 설정 시 **Reference Voltage Mode가 "Line to Neutral"로 ReadOnly(회색)** 변경.

> [이미지 Untitled 298.png] [다른 화면 예시] 분기 SETUP > MEASUREMENT > Load Mode. Mode "3-phase".

> [이미지 Untitled 299.png] [다른 화면/Shown] Mode "3-phase" 설정 시 Load Wiring에 **Phase Sequence(A-B-C) 항목 Shown**.

> [이미지 Untitled 300.png] [다른 화면/Shown] Mode "1-phase" 설정 시 Load Wiring에 **Phase Selection(AN, Feeder1/2/3) 항목 Shown**.

## 결정 배경

- Disabled/Hidden 정책 부재로 처리가 혼재. (Disabled: 항목 많으면 세로스크롤 필요 / Hidden: 항목 배치가 많이 변경됨)
