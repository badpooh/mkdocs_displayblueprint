# Policy-24-0028 · 설정변경팝업 최대입력자리수 정책

## 결정 사항

- 입력필드의 **최대입력자리수를 제한하지 않음**.
  - 예외: **A7300 Password 관련 화면은 4자리 제한**(「RELAY > SETUP > SYSTEM > Security > Password for Local Setup」, Password 해제 팝업).
- 예시-A2500: 입력필드 표시영역 초과 시 값의 **앞부분부터 표시하지 않음**(밀린 값은 보이지 않음):

> [이미지 image 67.png] [Tobe/A2500] PT Primary L-L Voltage 팝업(Range 1–999999) — 18자리 입력, 앞자리가 입력필드 밖으로 밀려 잘림(빨간 박스).

> [이미지 image 68.png] [Tobe/A2500] Device Name 풀키보드 팝업 — 긴 문자열 입력 시 앞부분("A")이 밀려 잘림(빨간 박스).

- **숫자값**: 설정범위 초과값 입력 후 Enter 시 **범위 최대값으로 Clipping**(예: 1~999999에서 1000000 입력 → 999999로 Entered).
- **문자열("Char")**: Enter 시 **앞에서부터 최대설정문자수까지 Clipping**(예: Device Name 30자 제한에 31자 입력 → 30자로 Entered, 설정조회화면엔 "20자+…" 표시 — Policy-24-0027 적용).

## 결정 배경

- 최대입력자리수 정책이 없어 장치·팝업별로 상이:

> [이미지 A2500_20240812_102811.png] [Asis/A2500] Min. Meas. Secondary L-N Volt. 팝업(Range 0–50) — **제한 없음**(20자리 입력됨).

> [이미지 A2500_20240814_092427.png] [Asis/A2500] Test Mode Timeout [min] 팝업(1–1440+Infinite) — **제한 없음**(25자리 입력됨).

> [이미지 A2500_20240812_103356.png] [Asis/A2500] IP Address 팝업(0–255, 4분할 입력) — 옥텟당 **3자리 제한**("999" 입력 상태).

> [이미지 A2500_20240813_111754.png] [Asis/A2500] Device Name 풀키보드 — **30자 제한**.

- A7300:

> [이미지 10.10.26.161_2024-08-12_19_39_11_M_S_ME_Voltage.png] [Asis/A7300 METER] Min. Meas. Secondary L-N Volt.(Range 1–10) — **10자리 제한**.

> [이미지 10.10.26.161_2024-08-13_17_54_35_M_S_CO_TestMode.png] [Asis/A7300 METER] Timeout [min](1–1440+Infinite) — **7자리 제한**.

> [이미지 10.10.26.161_2024-08-13_02_00_53_R_S_SY_Security.png] [Asis/A7300 RELAY] Password 팝업 — **4자리 제한**(●●●● 표시, 예외 유지 대상).

> [이미지 A3700N_20240812_171131.png] [Asis/A3700N] Min. Measured Secondary Voltage(Range 1~10) — **20자리 제한**.
