# Policy-23-0050 · "No Use" → "Not Used"

## 결정 사항

- **"No Use" 대신 "Not Used"** 용어 사용.

> [이미지 Untitled 301.png] [Tobe/A2500] Module 선택팝업 — 첫 항목 **"Not Used"**(선택 상태) + ID 101~115 모듈 리스트(DO/DIO/DO2), 하단 Channel 영역.

## 결정 배경

- "No Use"는 "쓸모 없다"는 의미라 다른 용어 필요.
- 「A7300-화면기획 > 진행대기사항」 화면용어검토(기술문서파트): ① Not in use(사용하지 않음) ② Unused(사용되지 않음) 제안. "Don't use"는 명령어 느낌이라 부적합. 타사 매뉴얼 확인 필요.
- 타사 매뉴얼 — RELAY:

> [이미지 Untitled 302.png] [참고/GE 850REF] Output Relay Availability 표 — I/O 할당 용어로 **"Assigned/Non-assigned"** Output Relays 사용(Available/Hidden 구분).

> [이미지 Untitled 303.png] [참고/GE 850REF] Virtual Outputs — "Virtual outputs **not assigned** for use are set to OFF (Logic 0)".

> [이미지 Untitled 304.png] [참고/Schneider SEPAM 80] Mimic-diagram Editor I/O assignment 창 — Assignment 컬럼, Assign/Delete 버튼.

> [이미지 Untitled 305.png] [참고/SEPAM 80] Logic input/output **assignment** 설명 — Assign 버튼으로 변수 연결, Delete로 해제.

> [이미지 Untitled 306.png] [참고/TOSHIBA GRE110] Signal List — 미할당 No.에 **"Not in use"** 사용.

> [이미지 Untitled 307.png] [참고/SEL-751] Modbus Map — 미할당 번지(400AH, 400EH)에 **"not used"** 사용.

- 타사 매뉴얼 — METER:

> [이미지 Untitled 308.png] [참고/GE PQM] Analog Outputs 설정 흐름도 — ANALOG OUTPUT 1 MAIN/ALT 디폴트 **"NOT USED"**.

> [이미지 Untitled 309.png] [참고/GE PQM] Switch Inputs(Setpoints Page 2) — SWITCH A NAME/FUNCTION 등에 "NOT USED" 사용.

> [이미지 Untitled 310.png] [참고/GE PQM] Modbus Memory Map F101 SWITCH INPUT STATUS — 미할당 비트에 **"Not Used"**.

> [이미지 Untitled 311.png] [참고/Schneider ION7650] Digital Output 모듈 설정표 — Port 설정값·디폴트 **"Not Used"**(Pulser·Calibration Pulser 동일).

- PowerDX2: 사용중/미사용중 "In Use/Not in Use", 사용/미사용 "Use/Don't Use".

> [이미지 Untitled 312.png] [참고/Google AI(Bard)] "Not in Use"/"Unused"/"Not Used" 차이 문답 — 표1·표2로 단어별 뉘앙스(현재 미사용 상태/한 번도 사용 안 됨/사용 가능) 비교 설명.
