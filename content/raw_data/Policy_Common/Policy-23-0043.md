# Policy-23-0043 · DI/DO 용어 수정 (Status, Debounce Time)

## 결정 사항

- **DI/DO "Status"** 용어 계속 사용:
  - Physical Status: "Closed"/"Open", Logical Status: "Off"(0)/"On"(1)
  - 장치 화면에는 **Physical Status만** 표시, 통신으로는 둘 다 제공(번지 맵핑).
- **"Debounce Time [msec]"** ← "Min. Pulse Width [msec]"에서 변경. 범위 0~255(step 1msec), 디폴트 10msec(계전기)/100msec(메터).

> [이미지 Untitled 416.png] [Tobe/A2550IO-DI] SETUP > I/O > DI [DI.1] — Polarity Normal, **Debounce Time [msec] 100**, Event Trigger Disable, Hold Time 3,600, Control "Counter Reset".
