# Policy-25-0005 · "Firmware Update" 팝업 추가

## 결정 사항

- 펌웨어 업데이트 동작 시 "Firmware Update" 팝업 표시. 동작 시점: 파일 수신 완료 후 장치 업데이트 수행 시점. 표시: **Loading 아이콘(.gif) + 경고 문구**(시간/진행률 없음).

> [이미지 image 33.png] [Tobe/A7300] Firmware Update 팝업 — "Updating Firmware... / Do not power off the device." + 로딩 닷(초록 테마).

> [이미지 image 34.png] [Tobe/A2500] 동일 팝업(파랑 테마), HOME Dashboard 위 표시.

> [이미지 image 35.png] [Tobe/A3700N] 동일 팝업.

## 결정 배경

- 업데이트 중 실제 동작과 무관한 쓰레기 값이 화면에 표시될 우려 → 사용자 오해 방지 필요. 제품별 표시 현황 상이:

| 제품 | 표시 현황 |
| --- | --- |
| A2500D(OLD) | "Firmware Update" 팝업(A2500D FW에 한함) |
| A2500D(Renewal) | "Firmware Update" 팝업(항상) |
| A2750D[C/CI] | "Update" 화면 |
| A2700D / A3700N / A7300 | 없음 |

> [이미지 20250721_091927.png] [참고/A2500D Renewal] FW 업데이트 시 "Updating firmware... Do not power off" + 로딩 닷 팝업(TEST 모드 화면 위).
