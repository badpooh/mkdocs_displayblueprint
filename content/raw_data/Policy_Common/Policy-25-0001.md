# Policy-25-0001 · 동일 메뉴 버튼 재입력 시 화면 전환

## 결정 사항

- 동일 버튼 입력 시 **해당 버튼 화면의 디폴트 상태로 전환**(A7300과 동일 방식).
  - **주메뉴 재선택**: 해당 주메뉴의 디폴트 상태(부메뉴 최상단 화면)로 전환.

> [이미지 image 16.png] [예시①/A7300 METER] 기존 화면 — VOLTAGE > THD % 에서 **L-N·Max 버튼 선택** 상태(빨간 박스), VOLTAGE 주메뉴 표시(빨간 원).

> [이미지 10.10.26.161_2025-01-17_15_25_17_M_H_VO_RMS.png] [예시②/A7300] 「Voltage」 주메뉴 재선택 → **RMS(부메뉴 최상단) 디폴트 화면**(L-L)으로 전환.

  - **부메뉴 재선택**: 해당 부메뉴의 디폴트 상태로 전환.

> [이미지 image 17.png] [예시①/A7300] THD % 화면 L-N·Max 선택 상태(THD % 부메뉴에 빨간 원).

> [이미지 10.10.26.161_2025-01-17_15_25_11_M_H_VO_THD.png] [예시②/A7300] 「THD %」 부메뉴 재선택 → THD % **디폴트 상태(L-L)** 로 전환.

## 결정 배경

- 제품별로 동일 버튼 재입력 동작이 다름:
- **A2500**: 주메뉴 재선택 → 주메뉴 화면(RMS) 전환 / **부메뉴 재선택 → 동작 없음**.

> [이미지 image 18.png] [Asis/A2500 ①] HOME > VOLTAGE > THD % — L-N·Max 선택 상태(VOLTAGE 빨간 원).

> [이미지 20250117_150517.png] [Asis/A2500 ②] Voltage 주메뉴 재선택 → RMS 화면(L-L 디폴트)으로 전환.

> [이미지 image 19.png] [Asis/A2500 ①] THD % 부메뉴(빨간 원) — L-N·Max 선택 상태.

> [이미지 image 20.png] [Asis/A2500 ②] THD % 부메뉴 재선택 → **동작 없음**(L-N·Max 선택 그대로 유지).

- **A3700N**: 주메뉴 재선택 → **현재 부메뉴 화면 유지·디폴트화** / 부메뉴 재선택 → 부메뉴 디폴트화.

> [이미지 image 21.png] [Asis/A3700N ①] THD % Max — L-N·Max 선택(VOLTAGE 빨간 원), 값 9.3/18.0/18.3%.

> [이미지 image 22.png] [Asis/A3700N ②] Voltage 주메뉴 재선택 → **THD % 화면(L-L 디폴트)** 으로 전환(RMS 아님).

> [이미지 image 23.png] [Asis/A3700N ①] THD %(빨간 원) L-N·Max 선택 상태.

> [이미지 image 24.png] [Asis/A3700N ②] THD % 부메뉴 재선택 → THD % L-L 디폴트로 전환.

- **A7300**: 주메뉴 재선택 → RMS(주메뉴 디폴트) 전환, 부메뉴 재선택 → 부메뉴 디폴트 전환 — **본 정책의 기준**(위 결정 사항 예시와 동일 화면).
