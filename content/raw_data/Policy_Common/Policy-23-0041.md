# Policy-23-0041 · DI/DO Counter 범위·동작·저장여부

## 결정 사항

- 기존 제품은 현재 구현 유지(저장·순환표시 현장 요구 없었음; 정책수정 시 전 모듈 업데이트 필요)
- 새제품(A2550IO): **16bit, Overflow, 저장 안 함**. 메터 제품은 기존과 동일, 계전 제품은 특수상황 고려해 저장·표시 다르게.

| 제품 | Bit(범위) | Limit/Overflow | 비고 |
| --- | --- | --- | --- |
| A2550IO | 16 (0~65535) | Overflow | 저장 안 함 |
| A7300 | 16 (0~60001) | Limit (60001) | 모든 DI/DO Counter 저장, 60001은 "60000+"로 표시 |

## 결정 배경

- 「SETUP」에서 채널별 "Counter Reset" 가능. 동작횟수는 비휘발메모리에 저장 안 함(전원 Off 시 0 리셋). 단 일부 모델 저장: A2700은 "RUN" DO 채널만, A7300은 모든 DI/DO.

| 제품 | Bit(범위) | Limit/Overflow | 비고 |
| --- | --- | --- | --- |
| A2350TEMP | 16 (0~65535) | Overflow |  |
| A2550TEMP | 16 (0~65535) | Overflow |  |
| A2750 | 32 (0~2^32-1) | Overflow | RUN 관련 DO만 카운팅·저장 |
| A3700, A3700N | 16 (0~65535) | Limit (65535) |  |
