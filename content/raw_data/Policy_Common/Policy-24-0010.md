# Policy-24-0010 · EVENT 필터/정렬 표시·동작

## 결정 사항

**디폴트**: Date 정렬 내림차순, Date/Index 필터 From·To 체크 해제(시각·숫자는 필터링된 리스트의 Oldest/Newest·Index 최소/최대값, 리스트 없으면 팝업 열린 시각·0/9999). Index는 0~9999 순환이라 Oldest/Newest 판단 애매(예: Oldest 9999, Newest 998).

**표시** — 표 Header에 필터·정렬 표시:
- 정렬 아이콘: 정삼각형 황색=ASC / 역삼각형 황색=DESC / 직각삼각형 백색=NONE / 없음=정렬불가
- 필터: 열 제목 글자색 황색=적용, 백색=미적용

> [이미지 Untitled 412.png] [Tobe] 표 Header. **Date 열이 황색(필터 적용)+황색 역삼각형(내림차순)**, Index는 백색 직각삼각형(NONE).

> [이미지 Untitled 413.png] [Asis] 표 Header. Date/Index 모두 백색 — 필터·정렬 상태 구분 안 됨.

**동작**: Date와 Index 필터/정렬은 Exclusive. Date 정렬 시 동일 Date 행은 같은 방향으로 Index 2차 정렬. 필터/정렬 변경 시 기존 데이터로 적용(새로고침 X)·1페이지 이동·스크롤 맨 위·행 선택 해제. 동일 설정으로 Enter 시 변화 없음(SETUP 동일 컨셉). 같은 화면 내 설정은 RAM 저장, 화면 이동 시 디폴트(Date 필터/정렬)로 복귀.

## 결정 배경

- A7300에 추가된 필터·정렬의 표시/동작 설계가 불명확.

> [이미지 Untitled 414.png] [참고] **Date 필터/정렬 팝업**. From 체크(2022-12-24 00:00:00)/To 해제, 방향키+숫자패드, Sort Ascending/**Descending(선택)**, Enter/Cancel.

> [이미지 Untitled 415.png] [참고] **Index 필터/정렬 팝업**(Date 팝업과 동일 형식).
