# Policy-23-0032 · 이벤트 리스트 「Read All」 버튼 추가

## 결정 사항

- **NewEvent가 있는 부메뉴에만 「Read All」 버튼 Enable**(버튼은 항상 표시). NewEvent가 있으면 「All」 부메뉴에도 Enable.

> [이미지 Untitled 47.png] [Tobe/A2500] EVENT > MEASUREMENT > All — NewEvent 행(노랑 #FFD95E Highlight) 존재 시 우측 상단 **「Read All」 Enable**, 종아이콘 알림 001.

> [이미지 Untitled 48.png] [Tobe/A2500] Voltage 부메뉴 — NewEvent 있어 「Read All」 Enable.

> [이미지 Untitled 49.png] [Tobe/A2500] Current 부메뉴 — NewEvent 없어 **「Read All」 Disabled**(회색).

- 장치별 행 Highlight 바탕색 통일: **NewEvent 행 #FFD95E(노랑), 선택된 행(Tap) #74A4EB(파랑)**.

> [이미지 Untitled 50.png] [Tobe/A2500] 선택된 행 — 파랑(#74A4EB) Highlight, Trend·Read All 버튼 표시.

- 「Read All」 Tap 시 **확인팝업 없이 바로** 읽음 처리: 부메뉴에서는 해당 부메뉴 NewEvent만, 「All」에서는 해당 주메뉴 전체 NewEvent 처리. 처리 후 행 바탕색 기본색 변경, 버튼 Disable, 모든 메뉴에 NewEvent 없으면 이벤트 알림창(종아이콘) 숨김.

> [이미지 Untitled 51.png] [A2500] NewEvent 있을 경우 — 종아이콘(001) 표시 + 노랑 행 + Read All Enable.

> [이미지 Untitled 52.png] [A2500] 모든 메뉴 NewEvent 없을 경우 — **종아이콘 숨김**, Read All Disabled, 행 기본색.

- (2023/09/14) Policy-23-0049 "데이터뷰 제목 Full Name 표기"에 따라, PQ Curve·Trend 버튼 동시 표시 시 타이틀 폰트 축소 건은 적용 불필요:

> [이미지 Untitled 53.png] [참고/A2500] POWER QUALITY > SEMI-F47 — 선택 행에 **PQ Curve + Trend + Read All** 3버튼 동시 표시(타이틀 공간 부족 사례).

- A7300에 기구현(2023/05/11 기준)이나 일부 로직 수정 필요: NewEvent 행 Highlight, 행 Tap 시 읽음 처리(NewEventCount 1 감소), Read All Tap 시 전체 읽음 처리(확인팝업 없음).

> [이미지 Untitled 54.png] [Asis/A2500] NewEvent Highlight 없음, 선택 행 **#74A4EB(파랑)** — Read All 버튼 없음.

> [이미지 Untitled 55.png] [Asis/A3700 장치사진] POWER QUALITY 이벤트 — NewEvent 없음, 선택 행 **#FFD95E(노랑)**.

> [이미지 Untitled 56.png] [Asis/A7300 RELAY] Event Log: All — NewEvent 행 **#74A4EB(파랑)**, Read All 버튼 표시.

> [이미지 Untitled 57.png] [Asis/A7300 METER] PQ Event: All — 선택 행 **#FFD95E(노랑)**(A2500과 반대 배색).

> [이미지 Untitled 58.png] [Asis/A7300 장치사진] PQ Event — **부메뉴마다 Read All 표시**(NewEvent 없어도), Tap 시 주메뉴 전체 읽음 처리.
