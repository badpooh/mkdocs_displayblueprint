# Policy-23-0033 · SETUP 변경Highlight 표시 정책 보완

## 결정 사항

- **[이슈-1]** 변경Highlight된 항목을 Apply 전에 기존 저장값으로 되돌린 경우: 기존 저장값과 현재값을 비교해 **다를 때만 Highlight**, 같으면 기본색. (AccuraSM은 예외 — 현재 상태 유지)
- **[이슈-2]** 변경Highlight된 항목이 Apply 전에 다른 항목에 의해 ReadOnly/Disabled 되는 경우: **"변경Highlight"와 "ReadOnly"/"Disabled"를 같이 표시**. 장치는 설정"값"에 대해 기능 동작에 문제가 발생할 경우에만 관여.

> [이미지 Untitled 367.png] [참고/A2500 GUI Design Guide] CONTENTS BOX(BUTTON) 상태 디자인 — Normal/Pressed(남색)/Disabled/Disabled(Pressed)/Read Only/Read Only(Pressed). 변경Highlight는 "Pressed"와 동일 디자인.

> [이미지 Untitled 368.png] [참고/A7300(A3700N) GUI Design Guide] 동일 상태표 — 테마색(파랑/초록/노랑)별 Normal/Pressed/Disabled(Pressed)/Read Only(Pressed) 색 정의.

- 예시-ReadOnly) 「메인 > SETUP > MEASUREMENT > Voltage > Wiring」 변경:

> [이미지 Untitled 369.png] [A2500/①] 변경 전 기본화면 — Wiring 3P4W, Reference Voltage Mode "Line to Neutral"(Enabled).

> [이미지 Untitled 370.png] [A2500/②] Wiring 3P4W→**3P3W** 변경 — Wiring 변경Highlight(남색), Reference Voltage Mode는 자동으로 "Line to Line"+**ReadOnly(회색)** 동시 표시, Apply/Cancel 버튼 표시.

> [이미지 Untitled 371.png] [A2500/③] Wiring을 기존값 3P4W로 되돌림 — Wiring Highlight 해제, Reference Voltage Mode는 Enabled 복귀+**변경Highlight 표시**(②에서 자동 변경된 값 "Line to Line" 유지). Apply/Cancel 계속 표시.

- 예시-Disabled) 「분기 > SETUP > MEASUREMENT > Load Wiring > Load Mode」 변경:

> [이미지 Untitled 372.png] [A2500 분기 CM-3P-60A/①] 변경 전 — Load Mode "3-phase mode", Feeder1~3 Phase Selection **Disabled**(AN/BN/CN).

> [이미지 Untitled 373.png] [A2500/②] Load Mode → **1-phase mode** 변경 — Load Mode 변경Highlight, Feeder1~3 자동 Enabled, Phase Sequence는 Disabled로 전환.

> [이미지 Untitled 374.png] [A2500/③] Feeder1 Phase Selection AN→**NA** 변경 — Feeder1도 변경Highlight.

> [이미지 Untitled 375.png] [A2500/④] Load Mode를 기존값 3-phase mode로 되돌림 — Load Mode Highlight 해제, Feeder1~3 자동 Disabled(사용자 변경값 NA 유지), Feeder1은 **변경Highlight+Disabled 동시 표시**. Apply/Cancel 계속 표시.

## 결정 배경

- SETUP 모드에서 값 변경 항목에 Highlight 표시 시, 위 2개 상황에 대해 장치별 구현이 달랐음.
- 설문조사(「A2500-UI기획 > 정기회의-20230518」) 등을 거쳐 최종안 결정.
