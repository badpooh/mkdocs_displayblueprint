# Policy-23-0049 · 데이터뷰 제목 Full Name 표기

## 결정 사항

- 정책: 데이터뷰 내용을 포괄(확장성 고려)하는 제목을 **가능한 Full Name**으로 표기.
  - 기본: 부메뉴명을 Full Name으로(부메뉴는 한 묶음의 관련 내용 표시). **주메뉴명·모드명과의 중복은 제거**.
  - 예외: 콤보박스 사용 시 현재 선택된 콤보 리스트명을 Full Name으로.
- (2024/11/09 추가) **데이터뷰 우상단 버튼(Max/Min/Peak 등)에 의한 뷰제목 변경은 하지 않음** — 폰트 변경이 필요한 구현 부담, 버튼 토글로 Max 여부 판단 가능, 내용의 "시각" 정보로도 인지 가능.
- 예-A2550IO:

> [이미지 Untitled 88.png] [Tobe/A2550IO] HOME > I/O — 제목 **"Digital Input"**(기존 "DI"에서 수정), DI.1~11 상태·카운트.

> [이미지 Untitled 89.png] [Tobe/A2550IO] SETUP > I/O — 제목 **"Digital Input 1"**(~11, 기존 "DI [DI.1]"에서 수정) + DI.1 콤보.

- 예-A3700N DIO:

> [이미지 Untitled 90.png] [Tobe/A3700N] HOME > MODULE > DIO — 제목 **"Digital Input & Output"** + DI/DO 사이 **세로구분선** 추가(A4D2·A2D4에도 권장).

> [이미지 Untitled 91.png] [Tobe/A3700N] SETUP > MODULE > DIO(DI) — 제목 "Digital Input 1"(~8).

> [이미지 Untitled 92.png] [Tobe/A3700N] SETUP > DIO(DO) — 제목 "Digital Output 1"(~2).

- 예-A7300_RELAY:

> [이미지 Untitled 93.png] [Tobe/A7300 RELAY] HOME > I/O > DI — 제목 "Digital Input"(채널명 CB 52a/52b/Remote Selection 등).

> [이미지 Untitled 94.png] [Tobe/A7300 RELAY] HOME > I/O > DO — 제목 "Digital Output"(CB.On/Off + DO.1~6, Lockout DO 등).

> [이미지 Untitled 95.png] [Tobe/A7300 RELAY] SETUP > I/O > DI — 제목 "Digital Input 1" + Test 버튼·DI.1 콤보.

> [이미지 Untitled 96.png] [Tobe/A7300 RELAY] SETUP > I/O > DO — 제목 "Digital Output 1"(Operate Type/Type 등).

> [이미지 Untitled 97.png] [Tobe/A7300 RELAY] SETUP > PROTECTION > OCR — 제목 **"OCR1 (Inv.)"**(콤보 선택 리스트명 Full Name 표기 예).

> [이미지 Untitled 98.png] [Tobe/A7300 RELAY] SETUP > MOTOR — 제목 "THR"(콤보 THR).

> [이미지 Untitled 99.png] [Tobe/A7300 RELAY] SETUP > MOTOR — 제목 **"I²t (Thermal Stress)"**(콤보 I²t의 Full Name).

> [이미지 Untitled 100.png] [Tobe/A7300 RELAY] SETUP > SUPERVISION — 제목 **"Trip Coil Supervision"**(콤보 TCS의 Full Name).

> [이미지 Untitled 101.png] [Tobe/A7300 RELAY] 동일 — 제목 **"Close Coil Supervision"**(콤보 CCS).

- 「EVENT」 데이터뷰 제목도 다른 모드처럼 **"부메뉴명"만** 표시:

> [이미지 Untitled 102.png] [Tobe/A2500] EVENT > MEASUREMENT — 제목 "Measurement Event: All" → **"All"**.

- HOME/SETUP/EVENT 모드 간 "누설 전류" 등 표기 통일(메인·분기 모두): 「HOME > CURrent > Leakage」 "Leakage Current (ZCT)"→**"Leakage Current"**(ZCT 삭제) / "Leakage & Residual"→"Leakage & Residual **Current**" / SETUP·EVENT의 "Leakage"/"Residual"→"Leakage Current"/"Residual Current"(DC모드는 "Residual Current" 표기 예정) / 「EVENT > PQ > Transient」 부메뉴 "Transient"→**"Transient Volt."**, 제목 "Transient"→**"Transient Voltage"**.
