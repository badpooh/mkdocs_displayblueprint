# Policy-23-0023 · TDD Nominal Current 디폴트값·팝업·항목명 수정

## 결정 사항

- 항목명 통일: **「TDD Nominal Current [A]」**, **「TDD Reference Selection」**.

> [이미지 Untitled 436.png] [참고/A2300·A2500 통신매뉴얼] Register 1264 Rated current(범위 1–기본전류) / 1265 TDD reference selection(0: TDD nominal current 사용(default), 1: Peak demand 값 사용) / 1266 TDD nominal current(1265가 0일 때 사용, 범위 0·1–200, Default 0). 조국춘 코멘트: "TDD를 맨 앞에 붙이고, TDD reference selection으로 표기 통일". TDD ref. sel.은 현재 제품들은 1 디폴트 정책 변경.

- 항목설명·범위·디폴트 수정(통신매뉴얼 수정 필요):
  - 「Rated Current [A]」: 범위 1–기본전류, Default 기본전류(모듈별 정격전류) — 기존 문서의 "정격전류" 표기는 오류.
  - 「TDD Nominal Current [A]」: 「TDD Reference Selection」이 "0: TDD Nominal Current"일 때 사용되는 전류 TDD 기준값. 범위: 0(「Rated current」를 TDD 기준값으로 사용) / 1–정격전류. **Default "0: 「Rated current」"**(A2350과 동일).
- 변경팝업: 숫자패드팝업에 **"Rated Current" 체크버튼 추가** — "0"의 의미를 정확히 전달. (팝업 형태 수정은 Policy-23-0024 참고)

> [이미지 Untitled 437.png] [Tobe/A2500] TDD Nominal Current [A] 변경팝업 — Range 1–60 + **"Rated Current" 체크박스**(해제 상태), 입력필드 60. 체크 해제 시 범위 최대값(60)으로 표시되는 예.

> [이미지 Untitled 438.png] [Tobe/A2500] SETUP > MEASUREMENT > Current 설정조회 — TDD Nominal Current [A]에 숫자 대신 **"Rated Current"** 텍스트 표시(0 설정 상태). Rated Current 60, TDD Reference Selection "Peak Demand Current".

- 「입력필드」 상황별 표시 규칙:
  - 팝업 열릴 때 현재값 임시보관(Cancel 시 임시보관 값 복원, Enter 시 수정값 표시). 용어: "Entered"(Enter 후 Apply 전), "Applied"(Apply 후).
  - 범위 **외** 값으로 적용된 상태에서 범위 **내** 값으로 변경 시: 입력필드에 의미상 가까운 최소/최대값 표시(예: Rated Current 체크 해제 → 60).
  - 범위 **내** 값(예: 50)으로 적용된 상태에서 범위 **외** 값으로 변경 시: 실제값 표시(Rated Current 해제→체크 → 0), 다시 범위 내로 변경 시 임시보관 값(50) 표시.

> [이미지 Untitled 439.png] [Tobe/A2500] 동일 팝업 — **"Rated Current" 체크 상태**: 입력필드 0(회색), 숫자패드 전체 Disabled.

> [이미지 Untitled 440.png] [Tobe/A2500] 동일 팝업 — 체크 해제 후 임시보관 값 50 복원 표시.

## 결정 배경

- 동일 항목인데 제품별 항목명이 달라 통일 필요.
- "기본전류" 용어 정의 후 통신매뉴얼에 미반영된 상태:

> [이미지 Untitled 441.png] [Asis/A2500] SETUP > Current 설정조회 — TDD Nominal Current [A]가 숫자 "60"으로만 표시되어 0(=Rated current 사용)의 의미 전달 불가.

> [이미지 Untitled 442.png] [참고] Smartsheet-Accura 2500 Data Details — Offset 3 Rated current(범위 1–정격전류, Default 정격전류(모듈별 정격전류)) / 4 TDD reference selection(Default 1: Peak demand current) / 5 TDD nominal current(범위 0·1–정격전류, **Default 정격전류** ← 사용자 불편 원인).

- 디폴트가 "정격전류"면, 기본전류 60A에서 정격전류를 50A로 변경할 때 TDD Nominal Current도 별도로 0 또는 50으로 재수정해야 하는 불편 → 디폴트 "0: Rated current"로 해결.
- 참고: 숫자패드만 있는 다른 팝업들은 범위 밖 값 Enter 시 범위 최소/최대값으로 자동변환(통신 입력도 동일). 예: 범위 60~3600에서 0/빈칸→60, 5000→3600.
