# Policy-23-0011 · 사용자입력문자열 고정폭 폰트·글자수 통일

## 결정 사항

1. 사용자입력문자열 표시 항목 중 **가로 길이 제한된 곳만 고정폭 폰트(DejaVu Sans Mono) 사용**(최대 12자): A2500D 「HOME/SETUP > Dashboard > SYSTEM > Location」, 「HOME > StatusView > Feeder 정보」. 「SETUP > SYSTEM > Description」은 가로 여유 있어 기본 폰트(Roboto).
2. 사용자입력문자열 표시 화면의 글자수 통일(글자수 적은 SETUP 기준, HOME은 폰트 크게).
3. (2024/08/20) Policy-24-0027로 관련 정책 추가.

## 결정 배경

- 기본 Roboto(가변폭)는 "W"/"M" 등이 넓어 표시 가능 범위를 벗어남.

> [이미지 Untitled 276.png] [Asis/장치] A2500D SETUP > Description. Location "COMPRESSOR MOTOR" 입력.

> [이미지 Untitled 277.png] [장치] A2500D SETUP 요약. SYSTEM Location "OMPRESSOR MO…"처럼 잘려/말줄임 표시.

> [이미지 Untitled 278.png] [장치] A2500D HOME Dashboard. SYSTEM Location "OMPRESSOR MOTO…" — HOME 폰트가 커서 더 잘림.

> [이미지 Untitled 279.png] [참고] StatusView Feeder 정보. "1234567890123"(13자)와 "1234567890.."(말줄임) — 고정폭 폰트 글자수 비교(ID01 Svr.Rack, ID03 Feeder).
