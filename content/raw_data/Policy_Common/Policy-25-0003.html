Policy-25-0003 · 장치 재부팅 기능 추가

결정 사항

  • 재부팅 기능 추가
  • 동작 조건: 전면 물리버튼 4개(HOME/SETUP/EVENT/BACK) 10초 Long Press. 통신 명령 등 다른 경로 미지원.
  • 동작: POWER 제외 LED 소등·LCD Backlight Off 후 재부팅(제어전원 On/Off와 동일). 재부팅 시 하위 모듈은 재부팅 안 함(A2300→A2350, A2500→A2550, A3700→IO 모듈).
  • A2500: 기능 추가하되 적용시기 추후 재검토 (Kernel 단 기능, 디스플레이 분리형은 Main 재부팅 가능 여부 확인 필요)
  • A7300: 추후 재검토 (기존 Reboot 동작·MCU Reboot 여부·기능 유효성 검토 필요)

결정 배경

  • A2300·A3700OLD·A3700N은 물리버튼 4개 약 10초 Press로 재부팅 기능 있음(매뉴얼엔 없음, 통신 재부팅 불가, 하위 모듈은 재부팅 안 함). 원본에 장치별 재부팅 영상(A2300/A3700OLD/A3700N .mp4) 첨부.
  • A2500·A7300에 재부팅 기능 추가 요청 — 현장에서 차단기 동작 없이 장치만 재부팅 필요(통신 리셋, 화면 멈춤 해소 등)

제품별 적용 이력

(적용 이력 없음 — 적용시기 재검토)
