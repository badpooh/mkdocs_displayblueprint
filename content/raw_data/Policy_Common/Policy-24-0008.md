# Policy-24-0008 · EVENT 표 행 선택/해제 시 제목영역 처리

## 결정 사항

- **행 선택 시**: 「Refresh」·「Read All」·「콤보박스(페이징)」 버튼 숨김. 선택 이벤트 상세정보 버튼(「Detail Data」·「Trend」 등)을 **우측 정렬** 표시. 선택 상태에서 새 이벤트 발생해도 Refresh는 숨김 유지(단 우측 상단 종아이콘·숫자는 갱신).
- 자동 행 선택 해제: 메뉴(화면) 이동, 필터/정렬 변경
- 행 선택 상태에서 세로스크롤로 선택행이 안 보여도 상세버튼 표시·선택 상태 유지 (A3700N은 선택 해제로 처리되어 「RMS」 버튼 숨겨짐)

> [이미지 Untitled 243.png] [Tobe/A3700 RELAY] EVENT > FAULT RECORD > With Operate, **행 선택 해제** 상태 — Read All·페이징(1/1) 표시.

> [이미지 Untitled 244.png] [Tobe] **행 선택 시** — Event Log/Waveform/Trend 버튼이 우측 정렬 표시되고 Read All/Refresh/페이징 숨김(선택행 파랑 강조).

## 결정 배경

- 행 선택 시 Refresh 숨김 + 상세버튼이 「Read All」 좌측에 표시되어 데이터뷰 제목이 표시되지 못하는 상황 발생.

> [이미지 Untitled 245.png] [Asis/A7300 RELAY] EVENT > FAULT RECORD > With Operate, 행 선택 해제.

> [이미지 Untitled 246.png] [Asis/A7300] 행 선택 시 — 상세버튼(Event Log/Waveform/Trend)이 **Read All 좌측**에 표시되어 제목영역을 가림(문제 상황).
