Policy-25-0001 · 동일 메뉴 버튼 재입력 시 화면 전환

결정 사항

  • 동일 버튼 입력 시 해당 버튼 화면의 디폴트 상태로 전환(A7300과 동일 방식).
  • 주메뉴 재선택: 해당 주메뉴의 디폴트 상태(부메뉴 최상단 화면)로 전환.
  • 부메뉴 재선택: 해당 부메뉴의 디폴트 상태로 전환.

결정 배경

  • 제품별로 동일 버튼 재입력 동작이 다름:
  • A2500: 주메뉴 재선택 → 주메뉴 화면(RMS) 전환 / 부메뉴 재선택 → 동작 없음.
  • A3700N: 주메뉴 재선택 → 현재 부메뉴 화면 유지·디폴트화 / 부메뉴 재선택 → 부메뉴 디폴트화.
  • A7300: 주메뉴 재선택 → RMS(주메뉴 디폴트) 전환, 부메뉴 재선택 → 부메뉴 디폴트 전환 — 본 정책의 기준(위 결정 사항 예시와 동일 화면).

제품별 적용 이력

(적용 이력 없음)
