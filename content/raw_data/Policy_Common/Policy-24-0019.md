# Policy-24-0019 · 「SETUP」 항목명 통일 (VT/L-L/Mode/Function)

## 결정 사항

A3700N setup 항목 명칭 규칙 반영:
1. **Enable/Disable 선택만 존재** → 'Function' 개념. 설정 명칭으로 표기('Function' 미표기): RSTP, Storm Control, Summer Time. (단 계전요소는 "Function" 허용)
2. **Enable/Disable로만 안 나뉨** → 'Mode' 개념. 설정명칭+'Mode': Test Mode
3. 각 설정항목에 포함되는 **메뉴명 생략**(단 메뉴명과 항목명이 완전 동일하면 예외: Summer Time, Test Mode): Demand Power Type→「Demand > Power Type」, Demand Sync Mode→「Demand > Sync Mode」, NTP Sync Mode→「NTP > Sync Mode」

항목명 통일 안: VT Primary/Secondary L-L Voltage [V], Power Type, Sync Mode, Modbus TCP Time out [sec], RSTP, Storm Control, Test Mode, Summer Time

- **VT 관련**: 신규 모델(A7300 이후)은 VT 사용, 기존 모델은 PT 유지. 선간전압 설정임을 명확히:

| A2500, A3700N | A7300_METER |
| --- | --- |
| PT Primary L-L Voltage [V] | VT Primary L-L Voltage [V] |
| PT Secondary L-L Voltage [V] | VT Secondary L-L Voltage [V] |

## 결정 배경

- 「A7300-UI기획 > 정기회의-20220510」에서 "PT Primary/Secondary Voltage"를 선간전압 명확화 위해 "PT ... L-L Voltage [V]"로 변경 결정.

> [이미지 Untitled 133.png] [참고] 장선영 이메일(2021-07-19) "A3700N setup 항목 명칭 규칙". 규칙①(Enable/Disable=Function, 미표기) 예: Dip Function→Dip, Swell Function→Swell, Summer Time. Storm Control Mode→Storm Control.

> [이미지 Untitled 134.png] [참고] 동 이메일 규칙②(Mode 개념): TDD Reference→TDD Reference Mode, Mode→Test Mode. ③(메뉴명 생략): Demand Sync. Mode→Sync. Mode. 예외: Demand Power Type은 2300부터 써와서 "Type" 유지.

- 제품별 「SETUP」 항목명이 상이(예: A2500 "PT Primary Voltage", A7300 "VT Primary L-L Voltage"; "Power Type" vs "Demand Power Type"; "RSTP Mode" vs "RSTP" 등).
