# Policy-24-0012 · DI/DO Counter 숨김/표시 설정·카운팅 통일

## 결정 사항

- 기능 삭제 대신 **"숨김/표시" 설정 추가**. 항목명 "DI/DO Counter", 범위 **0: Not Displayed(디폴트) / 1: Displayed** — 표시 유무만 설정(카운터 기능·값 영향 없음, Not Displayed여도 카운팅 지속·통신 수집 가능).
- 카운팅 방법 고정: **"Open → Closed"** 시 1씩 증가.
- 설정 위치: A2500 「메인 > SETUP > SYSTEM > Locale」, A3700N 「SETUP > SYSTEM > Locale」, A7300 「RELAY > SETUP > I/O > Etc.」

> [이미지 Untitled 135.png] [A2500] "Not Displayed" 시 — HOME > I/O > DI(ID31), Open/Closed 상태만(카운트 숨김).

> [이미지 Untitled 136.png] [A2500] "Displayed" 시 — 각 DI에 카운트(DI.1=1, DI.2=2, DI.5=3, DI.6=65535) 표시.

> [이미지 Untitled 137.png] [A3700N] "Not Displayed" 시 — HOME > MODULE > DI 상태만.

> [이미지 Untitled 138.png] [A3700N] "Displayed" 시 — 카운트 표시.

> [이미지 Untitled 139.png] [A7300] "Not Displayed" 시 — RELAY > HOME > I/O > DI(CB 52a/52b/Remote Selection 등 채널명 표시), 카운트 없음.

> [이미지 Untitled 140.png] [A7300] "Displayed" 시 — 각 채널 우측에 카운트(0) 표시.

## 결정 배경

- 피앤티팀 VOC(JB루트): "A3700N-DI Pulse count"에 대한 현장 질문 많음("0"/"1"만 표시 시 의미 애매) — 표시 제외 요청. 반면 기능을 요구하는 현장도 있음 → 설정으로 해결.
- 초기 검토안은 0:Not Used/1:Closed→Open/2:Open→Closed/3:Both였으나 표시 유무만 설정으로 확정(「A7300-UI기획 > 임시회의-20240425」 수정).
