# Policy-23-0018 · TFT_LCD 숫자값·단위표시 정책

> 최초 결정 2023/03/30. (2024/07/25) 추가·수정 규칙 포함 전체 재정리, (2024/12/23) 기본 단위가 없는 계측값(GW 등) 자리수 표시 수정. Custom_LCD(A2300S)는 기존 형식 유지.

## 결정 사항 — 기본 규칙 (A3700_UserGuide Appendix C 정밀도/신뢰도 참고)

- 설정에 따른 값 변환(예: 온도 ℃/℉, AI User-Scaled의 High/Low Value)이 필요한 경우, **변환 후** 규칙 적용.
- **"Reading" Scale**: 기본 4자리 소수점 표시, 값에 따라 단위 변경. **"FullScale"**: 항목별 소수점 자리수·단위 고정.
- **"Leading 0" 미표시.** 예외(항목 범위·일반 표시 상황 고려):

> [이미지 Untitled 150.png] [예외/A2500·A3700N] 이벤트 알림창 — NewEventCount를 Leading 0 포함 **3자리 정수**("009")로 표시. (모든 이벤트 합산이라 4자리가 맞지만 일반적으론 3자리면 충분)

> [이미지 Untitled 151.png] [예외/A2500] 모듈 ID 표시창 — Leading 0 포함 **2자리 정수**("ID 32", A2550은 0~39 설정 가능). 단 일반 텍스트 레이블의 모듈 ID는 Leading 0 미표시.

> [이미지 Untitled 152.png] [예외/A3700N] 모듈 ID — Leading 0 없이 **1자리 정수**("ID 1", 모듈 0~9 설정 가능).

- 값이 "0"일 때는 **기본 단위**로 표시, "0"이 아닐 때는 최소 단위까지만 내려갈 수 있음.
- **모든 값의 1000자리에 콤마(",") 미표시**, 소수점은 "."(도트). 예외적으로 콤마 표시: ①전력량 ②GW(통신 수집 정수형). ("소수점 기호"·"1000자리 기호" 설정은 추후 Locale 설정으로 검토)

## 결정 사항 — 확장 규칙

- 계측값(HOME)은 자리수에 맞게 **반올림**(사사오입, Round half up). 단 **전력량은 내림**(예: 234,567 Wh → 234 kWh).

> [이미지 Untitled 153.png] [참고/Excel] ROUND/ROUNDDOWN/ROUNDUP 함수의 양수·음수(3.5/-3.5/2.5/-2.5) 처리 결과 비교표.

- 설정값(SETUP)은 DataType(Integer/Float)과 무관하게 자리수에 맞게 반올림하여 저장·표시(통신 설정 포함):

> [이미지 Untitled 154.png] [예/A2500 통신맵] 5103 Dip threshold — UINT16, 0.10% 단위(Range 10–990 = 1.0–99.0%). "80.05" 입력 시 80.1로 저장.

> [이미지 Untitled 155.png] [예/A2550 통신맵] 5405 Threshold — FLOAT, % 단위(범위 5.0–200.0, Default 80.0). Dip 예시와 동일하게 반올림.

- **한 화면 내 동일 종류(A·B·C·Avg/Tot) 값의 단위 통일**(Max 값의 단위 기준). 소수점 자리수는 각 값에 따라 변경(기본규칙). 단 **"Tot"는 5자리 허용**(정수부 4자리 + 소수점 1자리, 예: 3상 900.1 kW → Tot 2700.3 kW). 대표값으로 Tot/Avg만 표시되는 경우(예: HOME Dashboard)는 미적용(Reading 4자리 기본규칙). CMS_1P3F·CMS_1P2F, CM_3P 1-phase mode는 피더별 독립 동작이라 각각의 단위·소수점.
- 동일 종류 값이 모두 "0"이면 기본 단위(예: 0A×3 → "0.000 A"×3 / 50mA, 50mA, 0A → "50.00 mA", "50.00 mA", "0.000 mA").
- **표시값 종류별 Scale/기본 단위/최소 단위/단위 변경 표**: 전압(Reading4, V, V↔kV↔MV↔GV↔TV) / 전류(Reading4, A, 최소 A·특정조건 mA③, mA↔A↔…↔TA) / 전력(Reading4, kW, 최소 W, W↔…↔TW — P/Q/S 모두) / 전력량(정수, kWh/Wh 설정, 단위 고정 — P/Q/S 모두) / 주파수(FullScale 소수3, Hz 고정) / 잔류 전압④·잔류 전류⑤(전압·전류와 동일, GVT·GCT 포함) / ZCT 전류(계측범위 2A 이하⑥: 정수 mA 고정, 10mA 이하⑦: Reading4 mA) / SAL⑧(Reading4, mA) / 백분율(FullScale 소수1, %) / 온도(FullScale 소수1, ℃/℉ 설정) / ELD(ZCT 2A와 동일) / RTD(PT100 소수1·PT1000 정수, Ω — 온도 환산은 "온도"와 동일) / AI(0~20/4~20mA)&AO(FullScale 소수2, mA) / AI(User-Scaled)&AO(Reading4, 단위없음, SI 접두어 m↔""↔k↔M↔G↔T) / GW 실수형(AI User-Scaled와 동일) / **GW 정수형(최대 14자리 정수, "-"·콤마 포함, SI 접두어 없음, 1000자리 콤마 표시)** / GW(User-Scaled)⑨(Reading7, 설정 단위) / Demand 전류·전력(전류·전력과 동일) / Phasor Angle⑩(소수1, º) / PF Value⑪(소수3, 단위없음).
  - ① 기본 단위 = 값 "0"일 때 표시 단위(예: 유효전력 0 → "0.000 kW").
  - ② **Auto Unit Change**: 유효숫자 초과 시 큰 단위로(V→kV), 0에 가까워져 Leading Zero로 유효숫자 소모 시 작은 단위로(A→mA, 최소 단위까지). **전압/전류 단위 고정("V/A") 기능**(A3700N만 제공, 현장(삼성) 요청): 정수부 4자리 이상이면 소수점 미표시, 최대 6자리 정수까지 표시(7자리 이상은 Auto Unit Change로 동작하되 설정값은 V/A 유지), 최소/최대값에도 적용, 이벤트로그 값에는 미적용. 설정 위치 「SETUP > SYSTEM > Locale > Measurement Unit」(Auto Unit Change / V/A(디폴트)):

> [이미지 Untitled 156.png] [A3700N] SETUP > SYSTEM > Locale — **Measurement Unit "V/A"**(빨간 박스), Timezone/Temperature Unit/Energy Unit/Comm. Event Log와 함께.

> [이미지 Untitled 157.png] [A3700N] V/A 고정 시 전압 — RMS Voltage **"10023 V"** 등 최대 6자리 정수 표시(소수점 없음).

> [이미지 Untitled 158.png] [A3700N] V/A 고정 시 전류 — RMS Current **"249372 A"** 등 6자리 정수(OVER % 표시).

> [이미지 Untitled 159.png] [A3700N] 이벤트로그 전압값 — Swell 이벤트 Description "579.1**kV**" 등 **Auto Unit Change로만** 표시.

> [이미지 Untitled 160.png] [A3700N] 이벤트로그 전류값 — Current Over "249.4**kA**" 등 Auto Unit Change 표시.

  - ③ 전류는 "5A 이하 모델 & CT비 1:1 이하" 조건에서 최소단위 mA(A2500: CT 5A 모델 & CT비 1:1 이하 — 한국전기연구원 정밀도 참고시험용):

> [이미지 Untitled 161.png] [참고] 한국전기연구원(KERI) 시험 관련 메모 — "In=5A의 1/100 인가 시 0.2~0.5% reading 오차 확인 요청. 50mA가 0.050 A로 표시되면 끝자리 해상도가 2%라 부족 → mA 표시 필요."

  - ④⑤ 잔류 전압·전류의 정밀도는 FullScale로 언급되나 표현은 Reading 관점(A2550SAL의 잔류 전류는 SAL 규칙).
  - ⑥ 계측용 ZCT는 범위 0~2000mA로 제한적 → mA 정수. ⑦ 보호계전용 ZCT(A7300)는 2차측 0~10mA(정격 1.5mA) → mA 소수점(A2750 ZCT는 0~20A라 "잔류 전류"와 동일). ⑧ SAL은 Surge Arrester 누설전류 모듈(A2550SAL, 0~20mA, 잔류 전류도 3상과 동일 표시).
  - ⑨ GW(User-Scaled)는 AccuraSM 전용 — 단위 설정·Expression 반영 변환값, Reading 7자리(예 "123456.7"). |반올림값|>999999.9면 소수3자리 지수(1234567→1.235e6), <0.000000이면 0.000000.
  - ⑩ Phasor Angle 계측값은 -180.0~180.0(−0.0/0.0/±180.0 표시 가능), 설정값은 -360~360(-0은 0으로 표시, A7300 Demo Test: SETUP은 입력값 그대로, HOME은 -180~180 자동 변환 — 예: 설정 240 → 계측 -120.0).
  - ⑪ PF Value는 -1.000~1.000(±0.000, ±1.000 표시 가능). PF Angle 표시는 Policy-23-0030 참고.
- **음수 표시 규칙**(대상: AC 유효/무효전력·PF·Demand 전력·Net 유효전력량, DC 전압/전류/유효전력/Net 전력량, Phasor Angle, 온도, AI(User-Scaled)·GW): 절대값 취해 SI 접두어·반올림 처리 후 부호 복원. "-0.0"과 "0.0" 모두 표시 가능(예: 온도 실제값 -0.03℃ → "-0.0 ℃"). "-" 표시 공간 확보 필요.
- **모든 화면에 동일 규칙**(RMS·Fundamental·Residual·Symmetrical·Phasor·Harmonics·Waveform 등). Harmonics/Waveform 그래프 축 단위에도 적용 — 체크(선택)된 값들로만 규칙 적용(예: 전압 모두 0이면 Y축 "[V]", 전류+ZCT 혼합 체크면 "[A]"로 통일, ZCT만 체크면 "[mA]", 모두 해제면 Y축 미표시). AC/DC 계측(DC 모듈 포함) 동일 적용.
