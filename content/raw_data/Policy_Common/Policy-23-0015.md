# Policy-23-0015 · "ZCT Sub Trend" → "ZCT Trend Data Type"

## 결정 사항

- 항목명 변경: "ZCT Sub Trend" → **"ZCT Trend Data Type"**

> [이미지 Untitled 209.png] [Tobe] SETUP > MEASUREMENT > Current(ID01, CMZ-3P-60A). Rated Current 60, TDD Reference Selection Peak Demand Current와 함께 **"ZCT Trend Data Type = RMS"**(변경된 항목명), ZCT Min. Measured Current 10mA.

## 결정 배경

> [이미지 Untitled 210.png] [참고] 조국춘 이메일(2023-02-21) "[A2550CMZ] trend data type 용어 관련". ZCT 있는 장치가 1-cycle RMS trend data 제공 시 Rms/Fund.Rms 설정 항목명을 "ZCT trend data type"으로 제안. 통신맵 12번 "ZCT trend data type"(0:ZCT RMS / 1:ZCT FUND, 디폴트 0).
