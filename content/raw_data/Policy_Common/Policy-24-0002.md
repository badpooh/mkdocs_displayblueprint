# Policy-24-0002 · IP Address/Gateway 변경팝업 형태 수정

## 결정 사항

- TextField를 4개로 분리(각 TextField Tap 영역을 전체 사각형 영역으로). 숫자패드를 "Type 2"로 변경.

> [이미지 Untitled 255.png] [Tobe/A3700N] IP Address 변경팝업. **TextField 4분리(10/10/10/100)**, 숫자패드 **Type 2(4열)**, Range 0–255.

## 결정 배경

- 「A7300-UI기획 > 임시회의-20231219 추가안건」 요청(안창모): IP 변경팝업 TextField가 터치 영역인데 숫자 그룹 구분이 불명확하고 세로 크기 작아 오터치 빈번.

> [이미지 Untitled 256.png] [Asis] IP Address 변경팝업. "10.10.10.100"이 **한 줄 TextField**, 숫자패드 Type 1(3열).

> [이미지 Untitled 257.png] [참고] NUMBER PAD 3종(Type1/Type2/Type3 Password) — Type 2로 변경해 영역 확대하는 방안.
