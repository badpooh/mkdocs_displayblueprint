# Policy-24-0017 · Reference Voltage 항목·팝업 형태 통일

## 결정 사항

- 「SETUP > … > Reference Voltage」 항목값·변경팝업을 제품간 통일.
- A3700N·A7300_METER: 수정 불필요. **A2500 수정 필요** — "Reference Voltage Mode"와 "Reference Primary Voltage [V]"를 A3700N처럼 **하나의 항목("Reference Voltage [V]")에서 "2 in 1" 팝업**으로 설정.

> [이미지 Untitled.png] [Tobe/A2500] SETUP > MEASUREMENT > Voltage — 두 항목이 **"Reference Voltage [V] = Line-to-Line, 380"** 단일 항목(빨간 박스)으로 통합, Volt. Phase Selection 나란히 표시.

> [이미지 Untitled 1.png] [Tobe/A2500, Wiring 3P4W·1P3W] "2 in 1" 변경팝업 — 상단 **Line-to-Line/Line-to-Neutral 토글** + Range 1–999,999 숫자패드(둘 다 선택 가능).

> [이미지 Untitled 2.png] [Tobe/A2500, Wiring 3P3W] 동일 팝업 — **Line-to-Neutral Disabled**(Line-to-Line만 선택 가능).

> [이미지 Untitled 3.png] [Tobe/A2500, Wiring 1P2W] 동일 팝업 — **Line-to-Line Disabled**(Line-to-Neutral만 선택 가능).

## 결정 배경

- 제품별로 항목값·변경팝업 형태가 다름:
- A3700N(A7300_METER 동일): "Reference Voltage [V]" 한 항목에서 Mode와 Primary Voltage를 "2 in 1" 팝업으로 설정.

> [이미지 Untitled 4.png] [Asis/A3700N] SETUP > MEASUREMENT > Voltage — **"Reference Voltage [V] = Line-to-Line, 380"** 단일 항목(빨간 박스), Sliding Reference Voltage·Rotating Sequence 등과 함께.

> [이미지 Untitled 5.png] [Asis/A3700N] "2 in 1" 팝업(초록 테마) — Line-to-Line/Line-to-Neutral 토글 + 숫자패드.

- A2500: 두 항목을 각각 설정.

> [이미지 Untitled 6.png] [Asis/A2500] SETUP > Voltage — **"Reference Voltage Mode = Line-to-Line"과 "Reference Primary Voltage [V] = 380"가 별도 항목**(빨간 박스 2개).

> [이미지 Untitled 7.png] [Asis/A2500] Reference Voltage Mode 선택팝업 — Line to Line / Line to Netural 리스트.

> [이미지 Untitled 8.png] [Asis/A2500] Reference Primary Voltage [V] 숫자패드팝업 — Range 1–999,999.
