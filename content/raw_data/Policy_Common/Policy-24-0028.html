Policy-24-0028 · 설정변경팝업 최대입력자리수 정책

결정 사항

  • 입력필드의 최대입력자리수를 제한하지 않음.
  • 예외: A7300 Password 관련 화면은 4자리 제한(「RELAY > SETUP > SYSTEM > Security > Password for Local Setup」, Password 해제 팝업).
  • 예시-A2500: 입력필드 표시영역 초과 시 값의 앞부분부터 표시하지 않음(밀린 값은 보이지 않음):
  • 숫자값: 설정범위 초과값 입력 후 Enter 시 범위 최대값으로 Clipping(예: 1~999999에서 1000000 입력 → 999999로 Entered).
  • 문자열("Char"): Enter 시 앞에서부터 최대설정문자수까지 Clipping(예: Device Name 30자 제한에 31자 입력 → 30자로 Entered, 설정조회화면엔 "20자+…" 표시 — Policy-24-0027 적용).

결정 배경

  • 최대입력자리수 정책이 없어 장치·팝업별로 상이:
  • A2500:
  • A7300:
  • A3700N:

제품별 적용 이력

(적용 이력 없음)
