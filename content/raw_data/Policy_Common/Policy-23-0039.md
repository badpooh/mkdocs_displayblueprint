# Policy-23-0039 · 통신 설정값 변경 시 Disabled/Hidden 속성 변경 시점 정책

> 상황: 사용자가 보고 있는 장치 화면 항목을 통신(AccuraSM 등)으로 변경하고, 그 변경이 다른 항목의 값/속성에 영향을 주는 경우. (예: 「Wiring」을 "3P3W"로 변경하면 「Reference Voltage Mode」가 "Line to Neutral"로 고정·ReadOnly로 변경)

## 결정 사항

- 현재 화면의 주메뉴/부메뉴 구성이 **변경되는** 설정항목: 모드별 「Dashboard」로 화면 자동전환
- 현재 화면의 주메뉴/부메뉴 구성이 **유지되는** 설정항목: 항목의 값·속성을 바로 변경

## 결정 배경

- 통신 변경 시 Disabled/Hidden 시점이 장치마다, 같은 장치 내 항목마다 다름
  - A3700N 「SETUP」: 값은 즉시 변경되나 영향받는 항목 속성 변경 시점이 항목별로 다름(즉시 반영 예: Rotating Sequence Hidden / 화면 재진입 시 반영 예: C Voltage Polarity Disabled). 「HOME」: 일부 항목(예: Wiring)은 적용 시 「HOME > Dashboard」로 자동전환.
  - A2500 「SETUP」: 값 즉시 변경, 속성 변경 시점 미확인(예: Wiring 변경 시 「SETUP > Dashboard」 자동전환). 「HOME」: 일부 항목(Wiring, AC/DC Mode) 적용 시 「HOME > Dashboard」 자동전환.
