# 영문 띄어쓰기 규칙조사 (2023-03-30, 신윤섭)

> Policy-23-0020(괄호 띄어쓰기)의 근거 조사 자료.

## 출처별 결론

| 출처 | 표시 |
| --- | --- |
| 국립국어원 | 조례 없음 |
| 카카오 | 영문 (영문), 소괄호 앞 띄어쓰기 원칙 |
| 글로벌 번역원 | 영문 (영문), 소괄호 앞 띄어쓰기 원칙 |
| KCI 한국학술지인용색인 | 영문 (영문), 소괄호 앞 띄어쓰기 원칙 |

- 국립국어원: 국문(국문)/국문(Korean)/Korean(국문)은 맞음, 영문(영문) 띄어쓰기 조례 없음

> [이미지 Untitled 454.png] [참고/카카오엔터프라이즈] 문장부호 - **괄호는 한글에서는 앞말에 붙여 쓰고, 영어에서는 앞말에 띄어 쓴다**(한글/영어 규칙이 반대).

> [이미지 Untitled 455.png] [참고/글로벌번역원] 괄호와 띄어쓰기 - 한글 문서는 괄호 앞 붙임, **영문 문서는 parenthesis 앞 띄어쓰기 필수**. 괄호와 약어(예: OECD)는 처음 표기 시 Full Name + (약어).

> [이미지 Untitled 456.png] [참고/KCI] APA Publication Manual 기준. 괄호 띄어쓰기는 괄호 앞 영문/숫자는 띄어 쓰고, 국문은 붙여 씀. 괄호 뒤 조사는 붙여 씀.
