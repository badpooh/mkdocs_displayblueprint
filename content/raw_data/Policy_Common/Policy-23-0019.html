Policy-23-0019 · 「SETUP > … > Setup Exit Timeout [sec]」 기능 통일

결정 사항

  • 동작: 변경모드/조회모드 무관하게 Setup Exit Timeout 후 HOME모드로 전환. EVENT모드는 모드 유지(자동전환 없음).
  • 항목명: 「Setup Exit Timeout [sec]」
  • 범위·디폴트 (메인장치 LCD大 / 모듈장치 LCD小 구분):
  • 메인장치: 60~3600 (디폴트 600초)
  • 모듈장치: 5~60 (디폴트 30초) — A2550CM, A2750P
  • 수정불가(하드웨어 사양): A2350CT 10초/30초 고정
  • 설정항목은 LCD화면·통신맵에 공개
  • 적용 우선순위: 신제품 즉시 적용, 기존제품은 연구부 상황에 맞게
  • 제품별 상세 수정결정은 "Setup Exit Timeout 기능의 제품별 수정결정-20230330" 참고

결정 배경

  • 기술영업부(안창모) 요청(2023/03/17): A3700에 있던 "설정모드 자동종료(HOME 전환)" 기능이 A3700N·A2500에서 빠짐
  • 연구개발부 확인: A2700D 개발부터 LCD 큰 제품에는 "설정모드종료" 미적용(현재 모드 명확히 인지 가능). A7300은 계전요소 설정으로 인해 기능 있음.
  • 기술영업부 추가의견(2023/03/21~24): LCD 크기 무관 전 제품에 기능 필요, 변경/조회모드 모두 일정시간 후 HOME 전환, EVENT는 유지, 항목명 "Setup Exit Timeout [sec]"(A2550의 "LCD Setup Exit Timeout"에서 "LCD" 삭제), 통신맵 공개, 신규제품 즉시·기존제품도 점진 반영

제품별 적용 이력

적용버전 담당자 날짜 비고
A7300 v1.0.0 박동민 2024/03/18
