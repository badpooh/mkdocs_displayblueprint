# Policy-24-0016 · EVENT Dashboard 제품 간 통일

## 결정 사항

- **A2500 기준으로 통일**. 단 A2500도 일부 수정: 왼쪽(주메뉴) 버튼 1번 Tap → 오른쪽에 데이터 표시, 그 상태에서 한 번 더 Tap → 해당 주메뉴 화면으로 이동.
- A3700N도 차후 Dashboard 추가.

## 결정 배경

> [이미지 Untitled 281.png] [참고/A2500 메인] EVENT > Dashboard. 좌측 주메뉴(MEASUREMENT 40/100·POWER QUALITY 79/100·SYSTEM 1,000/1,000), 우측에 선택 주메뉴의 부메뉴별 NewEvent 수(막대·Lamp)와 최근 이벤트 시각. 장점: 정보량 많음 / 단점: 부메뉴 최대 6개, 전체 한눈 보기 어려움.

> [이미지 Untitled 282.png] [참고/A2500 분기] ID01(CMZ-3P-60A) EVENT Dashboard. MEASUREMENT 22/100, Current/Power/Leakage/Etc.

> [이미지 Untitled 283.png] [참고/A7300 RELAY] EVENT Dashboard **3열**(EVENT LOG 125/1000, FAULT RECORD 10 Records, SYSTEM 1000/1000 + 부메뉴별 수치). 장점: 한눈에 전체 / 단점: 부메뉴별 NewEvent·최근시각 표시 어려움, RELAY 부메뉴 5개 제한(Ranking 표시).

> [이미지 Untitled 284.png] [참고/A7300 METER] **1열**(POWER QUALITY 425/1000, Dip/Swell/SEMI F47/IEC 61000/ITIC) — 1개 주메뉴라 표현 어색.

- A7300이 형태를 달리한 이유: 설계 초기 부메뉴 7개(이후 4개로 축소), 확장성 고려해 Ranking 5개 표시, 전체 상황 한눈 보기, RELAY HOME/SETUP Dashboard가 세로 2~3열 형식.
