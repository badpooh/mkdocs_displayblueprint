# Policy-23-0036 · 채널 선택창 콤보박스팝업

## 결정 사항

- 채널 선택 편의를 위해 **콤보박스팝업 추가**. 채널 선택창 = 좌/우 화살표(이전/다음, 기존 동일) + 중간 콤보박스(Tap 시 팝업 열림 — 기능 추가).
  - 팝업에서 채널 버튼 Tap: 팝업 닫히고 해당 채널로 데이터뷰 변경 / 콤보박스 재Tap·기타 영역 Tap: 팝업만 닫힘 / 주·부메뉴 Tap: 기존 화면전환
- 채널 선택창은 리스트 수와 무관하게 모두 콤보박스로 통일. 팝업 내 리스트는 **세로 열거**(숫자형·문자형 모두).

> [이미지 Untitled 71.png] [Tobe/A2500] SETUP(ID101, IO-DI) DI[DI.1] 기본화면 — 우측 상단 **DI.1 콤보박스형 채널 선택창**.

> [이미지 Untitled 72.png] [Tobe/A2500] 콤보박스팝업 열림 — DI.1~DI.11 채널 버튼 세로 열거.

## 결정 배경

- 기존 채널 선택창(좌/우 화살표 + 중간 레이블·반응 없음)만으로는 채널 많을 때 불편.

> [이미지 Untitled 73.png] [Asis/A2550TEMP] SETUP > EVENT > Temperature — 화살표+TEMP 레이블(콤보박스 아님).

> [이미지 Untitled 74.png] [Asis/A3700N] SETUP > MODULE > DIO[DI.1] — 화살표+DI.1 레이블(빨강 강조).

> [이미지 Untitled 75.png] [Asis/A7300_RELAY] SETUP > PROTECTION > MOTOR(THR) — 화살표+THR 레이블(리스트: THR, I²t, LST, Lock, UCR, Restart Inhibit).
