# Policy-23-0029 · 이벤트 DO/LED/알림창 동작 정책

## 결정 사항

- 「이벤트 로그」를 **MJ/MN 구분**:
  - **Major(MJ)**: 주요 이벤트 — 이벤트 LED·알림창 동작, NewEvent 처리(NewEventCount 증감, 이벤트리스트 행 New 바탕색).
  - **Minor(MN)**: 일상(참고) 이벤트 — LED·알림창 동작 안 함, NewEvent 처리 안 함.
- 「이벤트 알림창」 = "종 아이콘 + NewEventCount". 「EVENT」 모드에서는 알림창 좌측에 「DO Reset」 버튼 표시(A2500·A3700N만 해당, A7300은 Fault Reset 별도) — Tap 시 확인팝업 후 모든 DO 출력 Reset. NewEventCount는 「EVENT > Dashboard」에서 부메뉴별 확인.
- 「이벤트 DO」·「이벤트 LED」·「이벤트 알림창」은 모두 이벤트 알림 범주로 **가능한 유사하게 동작**(설정항목도 유사). LED와 알림창은 동일 동작 `[정책 수정 예정]`. 추가수정: End 없는 이벤트(예: 모듈탈락)는 Self-Resetting 설정 시 → Continuous로 동작하도록 수정 필요.
- 용어: **"Operate"**(이벤트 시작 시 출력 동작), **"Reset"**(이벤트 종료 or 사용자 리셋 시 출력 해제). 사용자 Reset 명령(DI·버튼·통신)이 Reset 설정 조건보다 우선.
- 3요소 비교표: 이벤트 DO(DO Notification 연동 채널, Latch/Pulse, 개별 이벤트로 동작) / 이벤트 LED(전면 적색 물리 LED, 점멸, 모든 활성 이벤트 OR 신호) / 이벤트 알림창(TFT 화면 종 아이콘, 점등, LED와 동일). Reset 모드: Continuous(사용자 Reset), Self-Resetting(이벤트 종료 or 사용자 Reset), Definite Time(시작부터 설정시간 경과 or 사용자 Reset). 사용자 Reset 방법: DI 입력 / 화면(DO는 「EVENT > DO Reset」 버튼, LED·알림창은 NewEvent 모두 Read 시 = Read All) / 통신 명령.
- [A2500/A3700N/A7300] 이벤트 종류별 알림: 이벤트 로그는 모든 시작/종료 기록. 모듈 탈락/추가/ID충돌, 링 결선/해제는 ChainView Saved 상태와 무관하게 기록·알림. 단 **"모듈 추가"는 알림 안 함**(임시회의-20250710 결정). 알림(O) 항목: PQ 시작, 계측·전압연결 시작(A7300은 None), 모듈 탈락/링 결선·해제/ID 충돌, 서브모듈 탈락·추가. 알림(X): 각 종료, 연결 상태 저장, 통신 접속/해제, 설정 변경.

## 결정 배경

- 이벤트 LED와 알림창이 일부 이벤트에서 다르게 동작해 통일 필요. 장치별 유무: A2300/A3700은 LED만, A2500/A2700/A3700N은 둘 다(A2700은 EVENT/ALARM/FAULT LED별 알림창, Count 미표시), **A7300은 알림창만**(메터부·계전부 각각, 동시 발생 시 같이 표시).

> [이미지 Untitled 125.png] [참고/A7300] 메터·계전부 이벤트 동시 발생 화면 — 종 아이콘에 **R 999(주황)·M 999(초록)** 동시 표시.

- A2500 사용자가이드(Rev1.61) 참고:

> [이미지 Untitled 126.png] [참고/A2500 매뉴얼] LED 상태표시 — POWER(Green), **EVENT(Red): 계측·PQ·점검 필요 시스템 이벤트 발생 시 점멸**, ETHERNET/MODULE(Orange).

> [이미지 Untitled 127.png] [참고/A2500 매뉴얼] 통신·장치 제공 이벤트 항목표 — PQ(Dip/Swell/SEMI F47/IEC/ITIC)·전류·누설전류·사용자정의는 시작/종료 계측값 제공, **전압연결(전압개방·퓨즈소손)은 시작 계측값 없음**(시작이벤트 자체는 발생).

- NewEventCount 규칙 모호 사례: 로그 전용 이벤트 100개 + PQ 시작 1개 발생 시 알림창 숫자가 101인지 1인지 → MJ/MN 구분으로 해소.
