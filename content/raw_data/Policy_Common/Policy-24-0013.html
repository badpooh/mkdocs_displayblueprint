Policy-24-0013 [진행중] · "Demo Test Mode" 파형을 "Reference Voltage" 기준 생성

결정 사항

  • Demo 파형을 "Reference Voltage" 기준으로 생성
  • 파형 생성 시 "L-L"/"L-N" 설정도 확인. "L-L"이면 "L-N" 값으로 변환 적용 (예: L-L 380V → 장치는 L-N 220V로 Demo 파형 생성)
  • 파형 생성 기준 변경은 ①"Demo Mode" 내 항목값 변경 시 ②"Reference Voltage" 변경 시 모두 즉시 반영 (이전 "변경 시점 고정" 안은 A7300 v1.0.0 테스트에서 문제 야기되어 폐기)
  • A7300 v1.0.0.006 기준 FW 수정 전이며, 다음 버전 테스트로 유효성 확인 후 정책 수정 예정 (관련 회의: A2500-20240423, A7300-20240425)

참고: 장치별 "Demo Test Mode"·"Reference Voltage" 설정 위치
- A2500: 「SETUP > CONTROL > Test Mode」 / 「MEASUREMENT > Voltage > Reference Primary Voltage [V]」
- A3700N: 「SETUP > CONTROL > Test Mode」 / 「MEASUREMENT > Voltage > Reference Voltage [V]」
- A7300_METER: 「SETUP > CONTROL > Test Mode」 / 「MEASUREMENT > Voltage > Primary Reference Voltage [V]」

결정 배경

  • A2500·A3700N·A7300_METER가 Demo를 상전압 220V 고정 기준으로 생성 → A7300_METER처럼 Default(190V)가 낮으면 Test Mode에서 Dip 미발생

제품별 적용 이력

(적용 이력 없음 — 진행중)
