# Policy-23-0056 · "Remote Control Lock Mode" 항목 추가

## 결정 사항

- FND 외 TFT 장치에도 기능 추가, 장치 화면에서 설정 가능하도록. 「SETUP > NETWORK > Advanced」에 추가:

| 값 | 항목값 | 디폴트 | 비고 |
| --- | --- | --- | --- |
| 0 | Each Connection Lock | V | 공간 부족 시 "Lock" |
| 1 | Always Unlock |  | 공간 부족 시 "Unlock" |

> [이미지 Untitled 332.png] [Tobe/A2500] 메인 SETUP > NETWORK > Advanced. Modbus TCP Timeout/RSTP Mode/Storm Control Mode/Port 502·503 Map/RS-485 Map + **"Remote Control Lock Mode = Each Connection Lock" 추가**.

> [이미지 Untitled 333.png] [Tobe/A3700N] SETUP > NETWORK > Advanced. 동일하게 **Remote Control Lock Mode 추가**.

> [이미지 Untitled 334.png] [Tobe/A7300 MeterPart] SETUP > NETWORK > Advanced. Modbus Timeout/RSTP/Storm Control + **Remote Control Lock Mode 추가**.

- A3500F 「SETUP > COMM」 FND 표시: 항목명 3행 "rMt/CtrL/ModE", 항목값 "Lock"/"unLoc" (실표시 후 수정 가능).

## 결정 배경

> [이미지 Untitled 335.png] [참고] 안창모 이메일(2023-09-27) "A3700N, A2500 control lock 설정 기능 관련 문의" — 장치 화면에서 설정 가능하도록 추가 요청(A3700N·A2500·A3500).

> [이미지 Untitled 336.png] [참고] 조국춘·안창모 카톡(2023-12-01). FND 장치의 Remote Control Lock 설정 위치 논의 — 통신 관련 항목이므로 RS-485 쪽(COMM)에 위치, Demo 설정은 DISPLAY 쪽.

- 일부 장치는 통신으로만 설정 가능한 상황:

> [이미지 Untitled 337.png] [참고/A3500 Modbus] Control Lock Function Mode — setup access(13401), **mode(13402): 0 Use each connection lock / 1 Always unlock**.

> [이미지 Untitled 338.png] [참고/AccuraSM 3500] Setup > Remote Lock Setup. Mode 콤보(Each Connection Lock/Always Unlock) — 개발자모드에서 표시·설정.

- A2500·A3700N·A7300은 항목 없었음 → (2023/10/17, 김민제) 통신맵에 항목 추가.
