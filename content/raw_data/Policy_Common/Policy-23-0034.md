# Policy-23-0034 · EVENT Function 항목명 "Trigger" 통일

## 결정 사항

- 이벤트의 Function에 해당하는 항목명을 "Trigger"로 통일
  - 설정항목 다수인 경우: "Trigger" 표시 (Dip, Swell, Transient Volt., Temperature, Diff. Temp. 등)
  - 설정항목 1개인 경우: 이벤트명만 표시 (SEMI~, IEC~, ITIC, Fuse Fail, Phase Open 등)
- 「Dip」·「Swell」에서 타이틀과 동일한 공통 항목명 삭제 → 항목명: Trigger, Threshold [%], Hysteresis [%]

> [이미지 Untitled 329.png] [Tobe] SETUP > EVENT > Dip. **Trigger(Disable)**, Threshold [%] 90.0, Hysteresis [%] 2.0 (타이틀 "Dip"과 중복되던 항목명 제거).

> [이미지 Untitled 330.png] [Tobe] SETUP > EVENT > PQ Curve. 설정항목 1개씩이라 **이벤트명만 표시** — SEMI F47-0706(Disable), IEC 61000-4-11/34 Class 3(Disable), ITIC(Disable).

> [이미지 Untitled 331.png] [참고/A7300_METER] SETUP > EVENT > Dip. "Dip"(Enable)/Threshold 90.0/Hysteresis 1.0/3-Phase Dip(Enable).

## 결정 배경

- 이벤트 Function 항목명을 장치·메뉴마다 다르게 표기(A2500: 메인 Dip "Dip Function", Transient Volt. "Event Trigger", 분기 Current/Leakage "Trigger", Temperature/Diff. Temp. "Event Trigger"). 「Dip」·「Swell」에 타이틀 중복 항목명 표시됨.
