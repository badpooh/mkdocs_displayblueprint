Policy-24-0032 · Ethernet DHCP 화면 표시 정책

결정 사항

  • IP Address/Subnet Mask/Gateway 항목에 DHCP(Enable/Disable)에 따라 Static/Dynamic 정보 표시:
  • DHCP "0: Disable": Static 정보(Asis 동일).
  • DHCP "1: Enable": Dynamic 정보 + 3항목 ReadOnly 표시.
  • DHCP가 정상 동작하지 않는 경우(예: 통신연결 문제): IP·Gateway "0.0.0.0", Subnet Mask "255.255.0.0" 표시.
  • DHCP 변경 Entered 상태 표시: Disable→Enable 시 "0.0.0.0"(&255.255.0.0)+ReadOnly+Highlight, Enable→Disable 시 Static 값+Normal+Highlight. Highlight는 설정조회화면 대비 값이 변경된 항목에만 적용:
  • 예1) Disable→Enable(Entered)→다시 Disable: 1단계에서 4항목 모두 Highlight, 2단계에서 원상복구(Highlight 없음).
  • 예2) Disable→Enable인데 기존 Subnet이 이미 255.255.0.0인 경우: Subnet은 값이 같으므로 Highlight 없이 ReadOnly만.
  • 예3) Enable→Disable(Entered)→다시 Enable: Static 값으로 Highlight 후 원상복구.
  • 예4) Enable→Disable인데 Gateway Static 값이 Dynamic 값과 동일(10.10.10.1)한 경우: Gateway는 Highlight 없음.
  • 💡 A2500은 현재(2024/10/24) DHCP 기능 없음 — 추후 현장 요청 시 추가.

결정 배경

  • Asis: IP/Subnet/Gateway 항목이 DHCP 설정과 무관하게 Static 정보만 표시(Dynamic 정보는 화면 미표시):
