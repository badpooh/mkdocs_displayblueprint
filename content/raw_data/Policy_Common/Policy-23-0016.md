# Policy-23-0016 · Description 빈 값은 빈칸 표시

## 결정 사항

- 「SETUP > … > Description」 문자열입력 항목에 값 없으면 빈칸(공백)으로 표시. Asis: "-" 표시 → Tobe: 빈칸.

> [이미지 Untitled 216.png] [Tobe 예시] SETUP > SYSTEM > Description. Device Name "Accura 2500", Location "ROOTECH", Installed Date 1970-01-01, **Description 항목은 값 없어 빈칸**(주석: "값을 입력하지 않으면 빈칸(공백)으로 표시되어야 함").

## 결정 배경

- "-"는 값 표시가 불가한 상태를 의미하므로 빈칸 처리 필요. 「HOME > SYSTEM > Location」은 빈칸 표시라 일관성 없음.
