# Policy-23-0051 · 항목명 대소문자·하이픈 표기 통일

## 결정 사항

- 「공통연구자료 > 장치의 항목명 표기 규칙」을 따름. 수정 예시:

| 관련 화면 | Asis | Tobe |
| --- | --- | --- |
| Voltage | (동일) | "Line-to-Line", "Line-to-Neutral" |
| Demand | "Sub-Interval Time [min]" | "Subinterval Time [min]" |
| Demand | "Number of Sub-Interval" | "Number of Subintervals" |
| Power | "PF Value at No-Load" | "PF Value at No Load" |
| Description | "User-defined Info" | "User-Defined Info" |
| Load Mode | "1-phase", "3-phase" | "1-Phase", "3-Phase" |
| DO | "Self-resetting", "Definite-time" | "Self-Resetting", "Definite Time" |

## 결정 배경

- 대소문자·하이픈 표기가 장치와 통신매뉴얼마다 상이.

> [이미지 Untitled 393.png] [참고/A2550DCM 통신매뉴얼] DO hold mode(6608): 0 Continuous / 1 **Self resetting** / 2 **Definite time**.

> [이미지 Untitled 394.png] [참고/A2550TEMP 통신매뉴얼] DO hold mode(4515): "Self-resetting"/"Definite time" — 장치 화면("Self Resetting")과 표기 상이.

> [이미지 A3700N_20230905_185453.png] [참고/A3700N 설정화면] SETUP > MODULE > DIO[DO.1]. Type **"Pulse,Self-Resetting"**(하이픈+대문자) 표기.

> [이미지 A3700N_20230905_185459.png] [참고/A3700N 수정팝업] Type(Latch/Pulse/Energy Pulse) + Hold Mode(Continuous/**Self Resetting**/Definite Time) — 팝업은 하이픈 없는 표기로 상이.

> [이미지 Untitled 395.png] [참고/A3700 통신매뉴얼] Hold mode of DO channel(125): 0 Continuous / 1 **Self-resetting** / 2 **Definite-time**.
