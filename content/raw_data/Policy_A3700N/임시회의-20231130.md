# 임시회의-20231130

## 안건 1: 「HOME > SYSTEM > A3700 > Hardware Revision」 버전정보 표시

- 출처: 「A2500-UI기획 > 정기회의-20231130」에서 결정

### Asis (기존)

> [이미지 Untitled 98.png] [Asis] HOME > SYSTEM > A3700 Information 화면. Serial Number, MAC Address, Application Version(1.6.998.2), Kernel Version, Bootloader Version, **Hardware Revision = "00.20.00"**, MCU Firmware/Bootloader Version 표시. (Touch Type/CPU/MCU Hardware Revision 포함 형태)

### Tobe (검토안)

> [이미지 Untitled 99.png] [Tobe안] HOME > SYSTEM > A3700 Information 화면. **Hardware Revision을 "00.00"으로 축약** 표시하고 MCU Hardware Revision은 표시하지 않음. CPU Firmware Version(A1.6.998, K1.0.021, B1.0.010)·MCU Firmware Version으로 통합 표시.

**👉 결정: Asis 유지** (추후 1A 모델 개발 시 버전 표시 재검토)

## 안건 2: 「EVENT > 이벤트 리스트(표)」 선택행·NewEvent 행 Highlight 바탕색 통일

- 참고: Policy-23-0032 (이벤트 리스트 데이터뷰 우측 상단 「Read All」 버튼 추가)

### Asis (기존)

> [이미지 Untitled 100.png] [Asis] EVENT > MEASUREMENT 이벤트 리스트. Index/Type/Date/Description 표에서 **선택된 행(Tap한 Event)이 노란색(#FFD95E)**으로 강조. NewEvent 행 강조는 없음.

### Tobe (개선안)

- NewEvent 행: #FFD95E(노랑), 선택된 행: #74A4EB(파랑)으로 분리

> [이미지 Untitled 101.png] [Tobe] EVENT 이벤트 리스트. **NewEvent 행을 노란색(#FFD95E)**으로 강조.

> [이미지 Untitled 102.png] [Tobe] EVENT 이벤트 리스트. **선택된 행(Tap한 Event)을 파란색(#74A4EB)**으로 강조 — NewEvent 노랑과 색상 구분.

**👉 결정: Tobe 안으로 결정**
