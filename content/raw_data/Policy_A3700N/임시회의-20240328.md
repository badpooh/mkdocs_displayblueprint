# 임시회의-20240328

## 안건: 「SETUP > SYSTEM > Locale」 DI Pulse Count(DI/DO Counter) 관련 설정 추가

- 출처: 「A7300-UI기획 > 임시회의-20240425」에서 결정사항 수정됨, 「A2500-UI기획 > 임시회의-20240328」 참고

### Tobe (검토안)

- 「SETUP > SYSTEM > Locale」에 "DI/DO Counter" 항목 추가 — 표시 유무 및 카운팅 방법 설정. 범위: 0=Not Used(디폴트), 1=Closed→Open, 2=Open→Closed, 3=Both

> [이미지 Untitled 54.png] [Tobe] SETUP > SYSTEM > Locale 화면. 기존 항목(Timezone 540, Temp Unit Celsius, Measurement V/A, Energy kWh, Comm. Event Log Enable, DI/DO Closed Status Color Red)에 더해 **"DI/DO Counter = Not Used" 항목이 추가**됨. 좌측 부메뉴에 CB도 존재.

- 「HOME > MODULE > DI」에서 "DI/DO Counter" 설정에 따른 화면 표시

> [이미지 Untitled 55.png] HOME > MODULE > DI(Digital Input, ID1) 화면. DI.1~DI.12의 Open/Closed 상태만 표시(DI.5·DI.6은 Closed). **DI/DO Counter "0:Not Used" 설정 시 카운터 값이 숨겨진 상태.**

> [이미지 Untitled 56.png] HOME > MODULE > DI 화면. 동일 화면이나 **각 DI 옆에 카운트 값(DI.1=1, DI.2=2, DI.5=3, DI.6=65535 등)이 표시**됨. DI/DO Counter를 Not Used 이외로 설정한 경우.

### 결정사항 👉

- 범위 및 디폴트 변경: **0=No Display(디폴트), 1=Display** — 표시 유무만 설정(카운터 기능·값에는 영향 없음)
- "0:No Display"여도 카운팅은 계속 수행(통신으로 카운터값 수집 가능)
- 카운팅 방법은 고정: "Open → Closed"로 될 때 1씩 증가 (A3700N 현재 구현과 동일)

### 관련 Task

- A3700N-설계추가-Rev0.92~0.96 화면설계 (기존화면, KT요청사항)
