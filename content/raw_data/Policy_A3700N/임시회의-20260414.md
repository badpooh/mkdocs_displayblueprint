# 임시회의-20260414

> 아래 안건은 장치에 선반영(또는 반영 중)된 건으로, 장치 변경 내역 공유·기록을 목적으로 함.

## 안건 [01]: [DO Module] Type 설정에 따른 이벤트 연동 DO 동작 정책

### 참고 화면 (A3700N)

> [이미지 image 100.png] SETUP > EVENT > Dip 화면. Dip(Disable), Threshold(90.0%), Hysteresis(2.0%), 3-Phase Dip(Disable)과 함께 **DO Notification = ID2 DO(Ch.1)** 항목 강조 표시(이벤트 발생 시 DO 출력 연동 설정).

> [이미지 image 101.png] SETUP > MODULE > DO[DO.1] 화면. Polarity(Normal), **Type = Energy Pulse**(강조), Pulse Period(0.20s), Pulse On Time(0.10s), Energy Scale(0.35 kWh/Pulse) 등.

> [이미지 image 102.png] SETUP > MODULE > DO[DO.1] 화면. **Type = Single Pulse**(강조)로 설정된 상태. Energy Scale/Event Parameter는 비활성(회색).

> [이미지 image 103.png] Accura 3700 User Guide(Rev2_80_Korean_260403) 발췌. "이벤트 알림" 설명과 Caution 박스: **"이벤트를 연동하는 채널의 DO Type이 Energy Pulse 또는 Single Pulse로 설정되어 있는 경우에는 이벤트가 발생하여도 DO 출력이 동작하지 않는다."**

### Asis (문제점)

- DO Notification으로 설정된 DO의 Type이 "Single Pulse"인 경우, 장치 동작과 User Guide 내용이 상이함. ("Energy Pulse"는 일치)

| DO Type | 장치 (v1.8.1p1) | User Guide (Rev2_80) |
| --- | --- | --- |
| Energy Pulse | DO 출력 동작 안 함 | DO 출력 동작 안 함 |
| Single Pulse | DO 출력 **동작함** | DO 출력 동작 안 함 |

### Tobe

- "Single Pulse"인 경우 장치 동작과 매뉴얼이 일치하도록 **매뉴얼 수정**(Single Pulse 관련 "DO 출력 동작 안 함" 내용 삭제). 장치 동작은 유지.

| DO Type | 장치 (v1.8.1p1) | User Guide (수정후) |
| --- | --- | --- |
| Energy Pulse | DO 출력 동작 안 함 | DO 출력 동작 안 함 |
| Single Pulse | DO 출력 동작 | DO 출력 관련 내용 삭제 |

### 결정사항 👉

- Tobe 안으로 결정 (조전무님과 합의하여 매뉴얼 내용 삭제). 이후 A2500도 동일 진행.
