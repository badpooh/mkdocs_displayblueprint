## 안건 [02]: 「SETUP > MODULE > DO」 Type 설정 범위 추가 (선반영, v1.8.0p1)

> [이미지 image 23.png] [참고] 문부기 이메일(2025-10-22) "A3700N DO 제어기능 확장". DO Type 기존 0:Latch/1:Pulse/2:Energy Pulse → 변경 **+3:Single Pulse**. Single Pulse = 제어 시 단일 pulse 출력 후 자동 꺼지는 단발성 명령(On만 있고 Off 없음, On time 지나면 자동 종료, Period 완료 전 추가명령 무시, A3700 Old의 uncountable pulse를 pulse 개수 1로 고정).

> [이미지 image 24.png] [참고] A3700N Modbus Map - Circuit Breaker 영역. CB On/Off Control Status, **CB On/Off Control(빨강)** 명령 레지스터 정의.

- Asis: Type 0:Latch(디폴트)/1:Pulse/2:Energy Pulse. CB 제어 명령 시 Type을 "1:Pulse"로 자동 변경.
- Tobe: **3:Single Pulse 추가**. CB 제어 명령 시 Type을 "3:Single Pulse"로 자동 변경.

**👉 결정: Tobe 안으로 결정**

## 안건 [03]: 「SETUP > MODULE > DO」 AccuraSM에서 Remote 설정 시 장치화면 표시 수정 (선반영, v1.8.0p3)

> [이미지 image 25.png] [참고] 문부기 이메일(2025-08-29) "A3700N DO 로컬제어 차단기능 구현 관련". 영업부 의견: 로컬제어 차단 시 설정화면을 read-only로 바꾸는 것은 불필요(삼성 요청 기능 아님). → **로컬 제어가 차단돼도 설정화면은 사용 가능하도록 예정**. 단 polarity 설정 변경 등으로 DO 출력 유발 가능.

> [이미지 image 26.png] [참고] 안창모 회신(2025-09-01). "삼성 특화 기능이므로 일반 제공 기능에 변경이 있는 것은 맞지 않아 고려하지 않았으면" 한다는 의견.

> [이미지 image 27.png] [참고] 논의 정리 이메일(2025-09-01). 결론: **DO 로컬 제어 차단기능 구현, 활성화 시 설정화면은 그대로 두고 DO 제어 메뉴도 popup 뜨고 입력 가능하되 로컬 제어명령은 실행 단계에서 차단**.

> [이미지 image 28.png] [참고] 문부기 이메일(2025-11-03) "A3700N CB 제어기능 설정 시 DO의 로컬제어 제한방법". "A3700N DO remote only 설정기능.docx" 첨부.

> [이미지 image 29.png] [참고/AccuraSM] Setup > Module > DO Channel Setup. **Control Permission(빨강)** = Remote로 설정된 채널의 DO Local 제어를 차단하는 설정. Ch1=Remote, Ch2~6=Local & Remote.

> [이미지 image 30.png] [참고/A2500] SETUP > NETWORK > Ethernet. MAC Address가 **Read Only로 표시 구현**(회색 비활성, 빨강 강조).

> [이미지 image 31.png] [참고/A3700N] SETUP > NETWORK > Ethernet. MAC Address **Read Only 표시 미구현 — Disabled와 동일하게 표시**(빨강).

> [이미지 image 32.png] [참고/A7300] METER > SETUP > NETWORK > Ethernet. MAC Address·USB IP Address가 **Read Only로 표시 구현**(빨강).

### Asis → Tobe

> [이미지 image 33.png] [Asis] SETUP > MODULE > DO[DO.1]. **Control Permission이 Remote여도 장치화면에서 Polarity/Type/Control 변경 가능**(Polarity Normal·Type Latch,Continuous·Control Open 항목 활성, 빨강 강조).

> [이미지 image 34.png] [Tobe] SETUP > MODULE > DO[DO.1]. **Control Permission이 Remote이면 전체 항목을 Disabled(회색)로 표시**하여 변경 불가(A3700N은 Read Only 표시 미구현이라 Disabled와 동일 표시).

- 대상: DO, DIO, A4D2, A2D4, DC, ELD의 DO 채널 모두 해당.
