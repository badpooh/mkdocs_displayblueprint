# 임시회의-20231207

## 안건: "Remote Control Lock Mode" 설정 항목 위치 변경

- 「SETUP > NETWORK > Advanced」의 "Remote Control Lock Mode" 설정 항목 위치 변경
- 출처: 「A2500-UI기획 > 정기회의-20231207」에서 결정
- 참고: 「A2500-UI기획 > 정기회의-20231026」에서 "Remote Control Lock Mode" 설정 항목이 처음 추가됨

### Tobe (개선안)

- 「SETUP > CONTROL > Security」 부메뉴 삭제

> [이미지 A3700N_20231205_022114.png] [Tobe] SETUP > CONTROL 화면. 좌측 부메뉴가 Data Reset / Demand Sync / Test Mode 로만 구성되고 **"Security" 부메뉴가 삭제됨**. 우측은 Data Reset 항목(Demand, Peak Demand, Max/Min, Energy, Measurement Event, PQ Event, System Event, DI/DO Counter) 표시.

- "Remote Control Lock Mode" 항목을 「SETUP > NETWORK > Advanced」로 이동

> [이미지 Untitled 108.png] [Tobe] SETUP > NETWORK > Advanced 화면. Modbus Timeout(600s), RSTP(Enable), Storm Control(Disable), Port 502/503 Map, RS-485 Map 항목과 함께 **"Remote Control Lock Mode = Each Connection Lock" 항목이 이동되어 추가됨**.

### 결정사항 👉

- Tobe 안으로 결정 (Security 부메뉴 삭제 + Remote Control Lock Mode를 NETWORK > Advanced로 이동)
