## 안건 1: 「SETUP > MODULE > DI/DO」 "Counter Reset" 항목 처리

### Asis

> [이미지 Untitled 57.png] [Asis/DI] SETUP > MODULE > DI(Digital Input 1). Polarity Normal, Debounce Time 100ms, Event Trigger Disable, Hold Time 3600, **Control = Counter Reset**.

> [이미지 Untitled 58.png] [Asis/DO] SETUP > MODULE > DO(Digital Output 1). Polarity Normal, Type Latch·Continuous, **Control = Open**.

### Tobe

- DI: "Control"을 "Counter Reset"으로 변경(Command 1 타입, SETUP>CONTROL>Data Reset과 동일). DO: "Control"은 일반 설정항목(Tap 시 변경팝업, 범위 Open/Closed), "Counter Reset"은 Command 1 타입. Counter Reset을 Control 앞에 배치(DI·DO 공통항목).

> [이미지 Untitled 59.png] [Tobe/DI] SETUP > MODULE > DI. Control이 **"Counter Reset"(Command 1 타입)**으로 변경됨.

> [이미지 Untitled 60.png] [Tobe/DO] SETUP > MODULE > DO(상단). Type Latch·Continuous, **Counter Reset 항목이 Control 앞에 배치**.

> [이미지 Untitled 61.png] [Tobe/DO] SETUP > MODULE > DO(스크롤 하단). Control = Open(일반 설정항목).
