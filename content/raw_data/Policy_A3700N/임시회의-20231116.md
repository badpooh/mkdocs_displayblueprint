# 임시회의-20231116

## 안건: HOME > Dashboard > SYSTEM에 CB(차단기) ON/OFF 상태 표시 추가

- 관련 위치: 「HOME > Dashboard > SYSTEM」에 CB ON/OFF 상태(DI 채널 맵핑)를 표시
- 출처: 「임시회의-20231123」에서 최종 결정됨
- VOC: 피앤티팀 - "A3700N HOME 화면에 CB ON/OFF 상태 표시 추가" 요청

### Asis (기존 화면)

**HOME 대시보드 — CB 상태 없음**

> [이미지 Untitled 34.png] 기존 HOME 대시보드 화면. VOLTAGE(220.1V, L-L)/CURRENT(29.96A)/POWER(17.127kW, PF 0.865) 및 ANALYSIS/MODULE/SYSTEM 6분할 패널 구성. SYSTEM 패널에는 Location("ROOTECH")만 표시되고 **CB Status 항목이 존재하지 않음** — CB 상태 표시 추가 이전 상태.

> [이미지 Untitled 35.png] 기존 HOME > SYSTEM 상세 화면. 좌측 메뉴 Description/Local Time/A3700/Network State/Temperature 중 Description 선택. Device Name(Accura7300), Location, Installation Date(1970-01-01) 표시. CB 관련 항목 없음.

**SETUP — CB 설정 없음**

> [이미지 Untitled 36.png] 기존 SETUP > SYSTEM > Locale 화면. Timezone Offset(540min), Temperature Unit(Celsius), Measurement Unit(V/A), Energy Unit(kWh), Comm. Event Log(Disable) 설정만 존재. **DI/DO Closed Status Color 항목과 CB 부메뉴가 없음**.

### Tobe (개선안)

**HOME**

- 「HOME > Dashboard > SYSTEM」에 "CB Status" 항목 추가
  - 「SETUP > … > CB On DI」, 「SETUP > … > CB Off DI」를 둘 다 "Not Used"로 설정하면 표시되지 않음
  - 둘 중 1개만 "Not Used"이면 1개만 표시 (Lamp 위치는 변경되지 않고 유지)
- 「HOME > SYSTEM」 자체는 CB 상태표시 관련 수정사항 없음 (차후 "Logic Module"의 버전·동작상태 정보가 추가될 수 있음)

> [이미지 Untitled 37.png] [Tobe] HOME 대시보드. SYSTEM 패널의 Location 아래에 **"CB Status" 항목이 신규 추가**되어 On(빨강 강조)/Off 토글로 차단기 상태를 표시. CB ON 상태 예시.

> [이미지 Untitled 38.png] [Tobe] HOME 대시보드. 동일하게 CB Status가 추가된 화면이며, **Off(초록 강조)/On** 으로 차단기 OFF 상태를 표시한 예시. On/Off 두 상태의 색상 표현 비교용.

**SETUP**

- 「SETUP > SYSTEM > Locale > DI/DO Closed Status Color」 항목 추가
  - [A2500]에 선(先) 추가된 항목
  - IO 모듈의 DI/DO 상태 표시 시, Closed 상태의 Lamp 색상 설정
  - 「HOME > Dashboard > SYSTEM > CB Status」의 Lamp 색상과는 무관
- 「SETUP > SYSTEM > CB」 부메뉴 추가
  - 「HOME > Dashboard > SYSTEM > CB Status」의 Lamp 색상을 각각 설정
  - 「CB On DI」, 「CB Off DI」 설정 변경팝업은 「SETUP > … > DO Notification」 변경팝업과 동일 형태

> [이미지 Untitled 39.png] [Tobe] SETUP > SYSTEM > Locale 화면. 기존 항목에 더해 **"DI/DO Closed Status Color = Red" 항목이 신규 추가**됨(A2500에 선 추가된 항목이라는 주석 표기). Measurement Unit은 Auto unit change, Comm. Event Log는 Enable 상태로 표시.

> [이미지 Untitled 40.png] [Tobe] SETUP > SYSTEM > **CB 부메뉴(신규)** 화면. CB On DI = ID3 DI(Ch.4), CB Off DI = ID3 DI(Ch.12)로 DI 채널을 맵핑하고, CB On Closed Status Color = Red, CB Off Closed Status Color = Green으로 설정. HOME의 CB Status 램프 색상과 DI 입력 채널을 지정하는 설정 화면.

### 결정사항 👉

- **HOME: 시안 수정하기로 함**
  - 이번에는 「HOME > Dashboard」 디자인 시안만 영업에 전달 (KT와 협의 목적) — 담당: 손정훈(디)
  - 「HOME > Dashboard > SYSTEM > CB Status」 수정 필요:
    - 상태 표시를 더 크게 (Location 위치 변경 등)
    - CB 상태는 1개만 사용(나머지 "Not Used")하더라도 On/Off 둘 다 표시 — 1개 접점으로 On/Off 모두 표시하는 상황 고려
- **SETUP: 결정 유보**
  - 설정을 어디서 보유할지 차후 결정(실무자 검토 후):
    1. Logic 모듈이 색상 포함 관련 설정 모두 보유 (Logic 모듈 ↔ A3700N App 간 정보 교환 채널 추가)
    2. Tobe안처럼 A3700N이 설정 보유
