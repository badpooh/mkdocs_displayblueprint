# 임시회의-20250819

## 안건 [01]: 1A 모델 지원에 따른 「SETUP > MEASUREMENT > Current」 디폴트 설정

### 참고 화면/문서

> [이미지 image 5.png] [참고/A3700N] SETUP > MEASUREMENT > Current 화면 + 디폴트값 표. CT Primary Current(50), CT Secondary Current(5), Reference Current(50), **Min. Measured Current(20mA, 빨강 강조)**, TDD Reference Selection(Peak Demand Current), Curr. Phase Selection(ABC: I1-I2-I3).

> [이미지 image 6.png] [참고] Accura 3700 User Guide(Rev2_72) 전류입력. 정격전류 5A nominal(3700-5A)/1A nominal(3700-1A), 계측범위 5A=1~200%In·1A=5~200%In, **최소 계측 = 5 mA**(빨강).

> [이미지 image 7.png] [참고/A3500] Accura 3500 User Guide(Rev2_22) 전류입력. 정격전류 5A/1A, **최소 계측 = 5 mA**(빨강).

> [이미지 image 8.png] [참고/A7300] METER > SETUP > MEASUREMENT > Current 화면. CT Primary(5), CT Secondary(5), Reference(5), **Min. Measured Current(5mA, 빨강)**.

> [이미지 image 9.png] [참고/A7300] Accura 7300 Catalog(Rev1_61). **최소 계측 = CT 전류 5 mA**(빨강), GCT 접지전류 1mA.

### Asis (A3700N 기존)

- 1A 모델 미지원. 5A 모델 디폴트: CT Primary 50 / CT Secondary 5 / Reference 50 / **Min. Measured Current 20mA**.

### Tobe (1A 모델 지원 추가, 2안 비교)

| 항목 | 5A 디폴트 | 1A 디폴트 | 비고 |
| --- | --- | --- | --- |
| CT Primary Current [A] | 50 | 10 | 공통 |
| CT Secondary Current [A] | 5 | 1 | 공통 |
| Reference Current [A] | 50 | 10 | 공통 |
| Min. Measured Current [mA] | **1안: 5 / 2안: 20** | **1안: 5 / 2안: 20** | 쟁점 |

- 1안: 최소 계측 디폴트를 5mA로 수정 / 2안: 20mA로 Asis 유지

### 결정사항 👉 (Tobe 1안으로 결정)

- **최소 전류 계측 디폴트를 5mA로 통일**(1안). 전력량계 "천분의 1" 적산 조건 충족에 5mA가 타당하고, 최근 노이즈 문제도 개선됨.
- 설정은 항상 0부터 시작.
- 참고: 1A 모델 정밀 인증(전기연구원 등)은 1mA 필요 — 시험 시 별도 변경.
